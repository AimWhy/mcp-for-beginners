# Eksempel

Dette er et Python-eksempel for en MCP-server.

Dette modul viser, hvordan man implementerer en grundlæggende MCP-server, der kan håndtere fuldførelsesanmodninger. Det giver en mock-implementering, der simulerer interaktion med forskellige AI-modeller.

Sådan ser værktøjsregistreringsprocessen ud:

```python
completion_tool = ToolDefinition(
    name="completion",
    description="Generate completions using AI models",
    parameters={
        "model": {
            "type": "string",
            "enum": self.models,
            "description": "The AI model to use for completion"
        },
        "prompt": {
            "type": "string",
            "description": "The prompt text to complete"
        },
        "temperature": {
            "type": "number",
            "description": "Sampling temperature (0.0 to 1.0)"
        },
        "max_tokens": {
            "type": "number",
            "description": "Maximum number of tokens to generate"
        }
    },
    required=["model", "prompt"]
)

# Register the tool with its handler
self.server.tools.register(completion_tool, self._handle_completion)
```
