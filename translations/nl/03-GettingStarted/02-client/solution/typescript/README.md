# Deze voorbeeld uitvoeren

Het wordt aanbevolen om `uv` te installeren, maar het is niet verplicht, zie [instructies](https://docs.astral.sh/uv/#highlights)

## -1- Installeer de afhankelijkheden

```bash
npm install
```

## -3- Start de server

```bash
npm run build
```

## -4- Start de client

```sh
npm run client
```

Je zou een resultaat moeten zien dat vergelijkbaar is met:

```text

  type: 'text',
  text: 'Please review this code:\n\nconsole.log("hello");'
}
```
