## 分佈式架構

分佈式架構係指多個 MCP 節點一齊合作處理請求、共享資源，並提供冗餘。呢個方法透過節點之間嘅通訊同協調，提升系統嘅擴展性同容錯能力。

我哋嚟睇個例子，點樣用 Redis 作協調，實現分佈式 MCP 伺服器架構。
