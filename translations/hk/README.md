# 🚀 Model Context Protocol (MCP) 初學者課程

## **透過 C#、Java、JavaScript、Python 及 TypeScript 實戰範例學習 MCP**

## 🧠 Model Context Protocol 課程概覽

**Model Context Protocol (MCP)** 是一個先進的框架，旨在標準化 AI 模型與客戶端應用程式之間的互動。這套開源課程提供結構化的學習路徑，搭配實用的程式碼範例與真實案例，涵蓋熱門程式語言如 C#、Java、JavaScript、TypeScript 和 Python。

無論你是 AI 開發者、系統架構師，還是軟體工程師，本指南都是你掌握 MCP 基礎與實作策略的完整資源。

## 🔗 官方 MCP 資源

- 📘 [MCP Documentation](https://modelcontextprotocol.io/) – 詳盡的教學與使用指南  
- 📜 [MCP Specification](https://spec.modelcontextprotocol.io/) – 協議架構與技術參考  
- 🧑‍💻 [MCP GitHub Repository](https://github.com/modelcontextprotocol) – 開源 SDK、工具與程式碼範例  

## 參加 MCP Dev Days 2025 年 7 月 29-30 日

準備好投入兩天的深度技術分享、社群交流與實作學習，MCP Dev Days 是一場專注於 Model Context Protocol (MCP) 的線上活動，這個新興標準正連結 AI 模型與其所依賴的工具。

➡️ [報名 MCP Dev Days](https://developer.microsoft.com/en-us/reactor/series/S-1563/)

你可以在活動頁面 https://aka.ms/mcpdevdays 註冊觀看 MCP Dev Days，屆時可透過 YouTube 或 Twitch 直播觀看。所有內容均會錄製並於活動後上傳至 Microsoft Developer YouTube 頻道。示範程式碼也會同步開放於 GitHub。

### 活動詳情
- 日期：7 月 29 日（第一天）及 7 月 30 日（第二天）
- 時間：每日太平洋標準時間上午 9:00
- 地點：線上，隨時隨地皆可參與！

#### 第一天：MCP 生產力、開發工具與社群

專注於協助開發者將 MCP 融入開發流程，並慶祝 MCP 社群的活力。我們將邀請 Arcade、Block、Okta、Neon 等社群成員與合作夥伴，分享他們如何與微軟合作，打造開放且可擴充的 MCP 生態系。內容包含 VS Code、Visual Studio、GitHub Copilot 及熱門社群工具的實際示範，實用且具情境導向的開發流程，以及由社群主導的議題分享。

無論你是剛接觸 MCP，還是在使用 MCP 開發，第一天都會帶來靈感與實用技巧。

#### 第二天：自信打造 MCP 伺服器

專為 MCP 建置者設計。深入探討 MCP 伺服器的實作策略與最佳實務，並介紹如何將 MCP 整合至 AI 工作流程。

### 主題包括：

- 建立 MCP 伺服器並整合至代理體驗
- 以提示驅動開發
- 安全最佳實務
- 使用 Functions、ACA 及 API 管理等組件
- 註冊表對齊與工具支援（第一方與第三方）

無論你是開發者、工具建置者或 AI 產品策略師，這一天的內容都充滿打造可擴展、安全且具備未來性的 MCP 解決方案所需的洞見。
