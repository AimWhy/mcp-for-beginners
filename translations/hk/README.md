# 🚀 Model Context Protocol (MCP) 初學者課程

## **透過 C#、Java、JavaScript、Python 及 TypeScript 實作範例學習 MCP**

## 🧠 Model Context Protocol 課程概覽

**Model Context Protocol (MCP)** 是一個先進框架，旨在標準化 AI 模型與客戶端應用程式之間的互動。這個開源課程提供有系統的學習路徑，搭配實務程式碼範例和真實案例，涵蓋 C#、Java、JavaScript、TypeScript 及 Python 等熱門程式語言。

無論你是 AI 開發者、系統架構師或軟件工程師，本指南都是你掌握 MCP 基礎及實作策略的全面資源。

## 🔗 官方 MCP 資源

- 📘 [MCP 文件](https://modelcontextprotocol.io/) – 詳盡教學與使用指南  
- 📜 [MCP 規格](https://spec.modelcontextprotocol.io/) – 協議架構與技術參考  
- 🧑‍💻 [MCP GitHub 倉庫](https://github.com/modelcontextprotocol) – 開源 SDK、工具及程式碼範例

## 🎯 學習 MCP 前置條件

為了從呢個課程獲益最大，你應該有：

- 基本嘅 C#、Java 或 Python 知識
- 明白客戶端-伺服器模型同 APIs
- （可選）熟悉機器學習概念

3. 練習去建立真實嘅 MCP 應用程式
