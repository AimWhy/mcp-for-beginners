# 🚀 初學者終極指南：Model Context Protocol (MCP) 課程

## **透過 C#、Java、JavaScript、Python 及 TypeScript 實戰代碼學習 MCP**

## 🧠 Model Context Protocol 課程概覽

**Model Context Protocol (MCP)** 是一個先進框架，旨在標準化 AI 模型與客戶端應用程式之間的互動。這個開源課程提供結構化的學習路徑，配合實用的程式碼範例及真實應用案例，涵蓋 C#、Java、JavaScript、TypeScript 和 Python 等主流程式語言。

無論你是 AI 開發者、系統架構師或軟件工程師，本指南都是你掌握 MCP 基礎及實作策略的全方位資源。

## 🔗 官方 MCP 資源

- 📘 [MCP Documentation](https://modelcontextprotocol.io/) – 詳盡的教學及用戶指南  
- 📜 [MCP Specification](https://spec.modelcontextprotocol.io/) – 協議架構與技術參考  
- 🧑‍💻 [MCP GitHub Repository](https://github.com/modelcontextprotocol) – 開源 SDK、工具及程式碼範例

|--|--|--|
| **00 - MCP 入門** | 介紹 Model Context Protocol 及其在 AI 流程中的重要性，包括 MCP 是什麼、標準化的意義，以及實際應用案例和效益 | [Introduction](./00-Introduction/README.md) |

| **02 - MCP 安全性** | 辨識 MCP 系統中的安全威脅，安全實作的技術與最佳實踐 | [Security](/02-Security/readme.md) |
| **03 - MCP 快速入門** | 環境設置與配置，建立基本的 MCP 伺服器與客戶端，將 MCP 整合至現有應用程式 | [Getting Started](./03-GettingStarted/README.md) |
| **3.1 - 第一個伺服器** | 使用 MCP 協議建立基本伺服器，理解伺服器與客戶端的互動，並進行測試 | [First Server](./03-GettingStarted/01-first-server/README.md) |

| **3.7 - 測試你的伺服器** | 測試是開發過程中重要的一環，本節將指導你使用多種工具進行測試 | [Testing your server](./03-GettingStarted/07-testing/README.md) |
| **3.8 - 部署你的伺服器** | 如何從本地開發進入生產環境？本節將協助你開發及部署伺服器 | [Deploy your server](./03-GettingStarted/08-deployment/README.md) |
| **04 - 實務應用** | 使用多種語言的 SDK，除錯、測試與驗證，打造可重複使用的提示模板及工作流程 | [Practical Implementation](./04-PracticalImplementation/README.md) |
| **05 - MCP 進階主題** | 多模態 AI 工作流程與擴充性、安全擴展策略、MCP 在企業生態系統中的應用 | [Advanced Topics](./05-AdvancedTopics/README.md) |
| **06 - 社群貢獻** | 如何貢獻程式碼及文件，透過 GitHub 協作，社群驅動的增強與回饋 | [Community Contributions](./06-CommunityContributions/README.md) |
| **07 - 早期採用者見解** | 真實世界的實作案例與成功經驗，MCP 解決方案的建置與部署，趨勢與未來路線圖 | [Insights](./07-LessonsFromEarlyAdoption/README.md) |
| **08 - MCP 最佳實踐** | 效能調校與優化，設計容錯的 MCP 系統，測試與韌性策略 | [Best Practices](./08-BestPractices/README.md) |
| **09 - MCP 案例研究** | 深入探討 MCP 解決方案架構、部署藍圖與整合技巧，附帶註解圖表及專案導覽 | [Case Studies](./09-CaseStudy/README.md) |

## 🎯 學習 MCP 的先決條件

要充分利用呢個課程，你應該具備：

- 基本嘅 C#、Java 或 Python 知識
- 明白客戶端-伺服器模式同 API
- （可選）熟悉機器學習嘅概念
