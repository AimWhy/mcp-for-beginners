# 🚀 Model Context Protocol (MCP) 初學者課程

## **用 C#、Java、JavaScript、Python 同 TypeScript 嘅實戰代碼學 MCP**

## 🧠 Model Context Protocol 課程概覽

**Model Context Protocol (MCP)** 係一個先進框架，設計用嚟標準化 AI 模型同客戶端應用程式之間嘅互動。呢個開源課程提供有系統嘅學習路徑，配合實用代碼範例同真實應用案例，涵蓋 C#、Java、JavaScript、TypeScript 同 Python 等熱門編程語言。

無論你係 AI 開發者、系統架構師定係軟件工程師，呢份指南都係你掌握 MCP 基礎同實踐策略嘅全面資源。

## 🔗 官方 MCP 資源

- 📘 [MCP Documentation](https://modelcontextprotocol.io/) – 詳細教學同用戶指南  
- 📜 [MCP Specification](https://spec.modelcontextprotocol.io/) – 協議架構同技術參考  
- 🧑‍💻 [MCP GitHub Repository](https://github.com/modelcontextprotocol) – 開源 SDK、工具同代碼範例

## 🎯 學習 MCP 的先決條件

要充分掌握這個課程，你應該具備：

- 基本的 C#、Java 或 Python 知識
- 理解客戶端-伺服器模型及 API
- （可選）熟悉機器學習概念
