# 🚀 منهج بروتوكول سياق النموذج (MCP) للمبتدئين

## **تعلم MCP من خلال أمثلة عملية في C#، Java، JavaScript، Python، وTypeScript**

## 🧠 نظرة عامة على منهج بروتوكول سياق النموذج

يُعد **بروتوكول سياق النموذج (MCP)** إطار عمل متقدم مصمم لتوحيد التفاعلات بين نماذج الذكاء الاصطناعي وتطبيقات العملاء. يقدم هذا المنهج مفتوح المصدر مسار تعلم منظم، يتضمن أمثلة برمجية عملية وحالات استخدام واقعية، عبر لغات برمجة شائعة مثل C#، Java، JavaScript، TypeScript، وPython.

سواء كنت مطور ذكاء اصطناعي، مهندس نظم، أو مهندس برمجيات، فإن هذا الدليل هو مصدر شامل لإتقان أساسيات MCP واستراتيجيات تطبيقه.

## 🔗 الموارد الرسمية لـ MCP

- 📘 [توثيق MCP](https://modelcontextprotocol.io/) – دروس تفصيلية وأدلة المستخدم  
- 📜 [مواصفات MCP](https://spec.modelcontextprotocol.io/) – هيكل البروتوكول والمراجع التقنية  
- 🧑‍💻 [مستودع MCP على GitHub](https://github.com/modelcontextprotocol) – مجموعات تطوير البرامج مفتوحة المصدر، الأدوات، وأمثلة الكود  

## انضم إلينا في MCP Dev Days في 29-30 يوليو 2025

استعد ليومين مليئين بالرؤى التقنية العميقة، والتواصل المجتمعي، والتعلم العملي في MCP Dev Days، وهو حدث افتراضي مكرس لبروتوكول سياق النموذج (MCP) — المعيار الناشئ الذي يربط نماذج الذكاء الاصطناعي بالأدوات التي تعتمد عليها.

➡️ [سجل في MCP Dev Days](https://developer.microsoft.com/en-us/reactor/series/S-1563/)

يمكنك متابعة MCP Dev Days عبر التسجيل في صفحة الحدث: https://aka.ms/mcpdevdays. من هناك، ستتمكن من الانضمام للبث المباشر على YouTube أو Twitch. جميع المحتويات مسجلة وستكون متاحة لاحقًا على قناة Microsoft Developer على YouTube. كما سيكون الكود المصدري للعروض التوضيحية متاحًا على GitHub.

- الوقت: 9:00 صباحًا بتوقيت المحيط الهادئ يوميًا
- المكان: عبر الإنترنت – انضم من أي مكان!

#### اليوم الأول: إنتاجية MCP، أدوات التطوير، والمجتمع:

يركز على تمكين المطورين من استخدام MCP في سير عملهم والاحتفاء بمجتمع MCP الرائع. سينضم إلينا أعضاء المجتمع وشركاء مثل Arcade، Block، Okta، وNeon لنرى كيف يتعاونون مع Microsoft لبناء نظام MCP مفتوح وقابل للتوسع. عروض حية عبر VS Code، Visual Studio، GitHub Copilot، وأدوات المجتمع الشهيرة  
سير عمل تطوير عملي قائم على السياق  
جلسات ورؤى يقودها المجتمع  
سواء كنت تبدأ للتو مع MCP أو تبني به بالفعل، اليوم الأول سيضع الأساس بالإلهام والنصائح القابلة للتطبيق.

#### اليوم الثاني: بناء خوادم MCP بثقة

مخصص لبناة MCP. سنغوص في استراتيجيات التنفيذ وأفضل الممارسات لإنشاء خوادم MCP ودمج MCP في سير عمل الذكاء الاصطناعي الخاص بك.

### المواضيع تشمل:

- بناء خوادم MCP ودمجها في تجارب الوكلاء
- تطوير قائم على المطالبات
- أفضل ممارسات الأمان
- استخدام اللبنات الأساسية مثل Functions، ACA، وإدارة API
- مواءمة السجلات والأدوات (1P + 3P)

إذا كنت مطورًا، أو باني أدوات، أو استراتيجي منتجات ذكاء اصطناعي، فهذا اليوم مليء بالرؤى التي تحتاجها لبناء حلول MCP قابلة للتوسع، آمنة، وجاهزة للمستقبل.
