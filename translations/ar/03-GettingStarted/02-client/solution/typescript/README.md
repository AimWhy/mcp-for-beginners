# تشغيل هذا النموذج

يوصى بتثبيت `uv` ولكن ليس بالضرورة، انظر [التعليمات](https://docs.astral.sh/uv/#highlights)

## -1- تثبيت المتطلبات

```bash
npm install
```

## -3- تشغيل الخادم

```bash
npm run build
```

## -4- تشغيل العميل

```sh
npm run client
```

يجب أن ترى نتيجة مشابهة لـ:

```text
Prompt:  {
  type: 'text',
  text: 'Please review this code:\n\nconsole.log("hello");'
}
Resource template:  file
Tool result:  { content: [ { type: 'text', text: '9' } ] }
```
