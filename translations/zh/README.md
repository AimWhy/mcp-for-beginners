## 🧠 Model Context Protocol 课程概览

**Model Context Protocol (MCP)** 是一个前沿框架，旨在标准化 AI 模型与客户端应用之间的交互。这个开源课程提供了结构化的学习路径，配备实用的编码示例和真实案例，涵盖 C#、Java、JavaScript、TypeScript 和 Python 等主流编程语言。

无论你是 AI 开发者、系统架构师还是软件工程师，本指南都是你掌握 MCP 基础知识和实现策略的全面资源。

- 📘 [MCP 文档](https://modelcontextprotocol.io/) – 详细教程和用户指南

## 🎯 学习 MCP 的先决条件

- C#、Java 或 Python 的基础知识
