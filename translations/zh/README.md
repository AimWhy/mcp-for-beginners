# 🚀 面向初学者的模型上下文协议（MCP）课程

## **通过 C#、Java、JavaScript、Python 和 TypeScript 的实战代码示例学习 MCP**

## 🧠 模型上下文协议课程概览

**模型上下文协议（MCP）** 是一个前沿框架，旨在规范 AI 模型与客户端应用之间的交互。这个开源课程提供了结构化的学习路径，配有实用的代码示例和真实场景案例，涵盖 C#、Java、JavaScript、TypeScript 和 Python 等主流编程语言。

无论你是 AI 开发者、系统架构师还是软件工程师，本指南都是你掌握 MCP 基础知识和实现策略的全面资源。

## 🔗 官方 MCP 资源

- 📘 [MCP 文档](https://modelcontextprotocol.io/) – 详细的教程和用户指南  
- 📜 [MCP 规范](https://spec.modelcontextprotocol.io/) – 协议架构和技术参考  
- 🧑‍💻 [MCP GitHub 仓库](https://github.com/modelcontextprotocol) – 开源 SDK、工具和代码示例  

## 加入我们参加 2025 年 7 月 29-30 日的 MCP 开发者日

准备好参加为期两天的深入技术分享、社区交流和动手学习活动——MCP 开发者日，这是一个专注于模型上下文协议（MCP）的线上活动，致力于推动这一连接 AI 模型及其依赖工具的新兴标准。

➡️ [注册 MCP 开发者日](https://developer.microsoft.com/en-us/reactor/series/S-1563/)

你可以通过活动页面 https://aka.ms/mcpdevdays 注册观看 MCP 开发者日直播。届时可通过 YouTube 或 Twitch 观看直播。所有内容都会录制，并在微软开发者 YouTube 频道上回放。演示的源代码也会在 GitHub 上发布。

### 活动详情
- 日期：7 月 29 日（第一天）和 7 月 30 日（第二天）
- 时间：每天上午 9:00（太平洋时间）
- 地点：线上，随时随地参与！

#### 第一天：MCP 生产力、开发工具与社区

重点是帮助开发者将 MCP 融入开发流程，庆祝精彩的 MCP 社区。届时将邀请 Arcade、Block、Okta 和 Neon 等社区成员与合作伙伴，展示他们如何与微软协作，共同打造开放且可扩展的 MCP 生态系统。内容包括 VS Code、Visual Studio、GitHub Copilot 及流行社区工具的真实演示，实用且以上下文为驱动的开发工作流，以及社区主导的分享和见解。

无论你是 MCP 新手还是已经在使用 MCP，第一天都将带来启发与实用的干货。

#### 第二天：自信构建 MCP 服务器

面向 MCP 构建者。深入探讨 MCP 服务器的实现策略和最佳实践，以及如何将 MCP 集成到你的 AI 工作流中。

### 议题包括：

- 构建 MCP 服务器并集成到代理体验中
- 基于提示的开发
- 安全最佳实践
- 使用 Functions、ACA 和 API 管理等构建模块
- 注册表对齐与工具支持（第一方与第三方）

如果你是开发者、工具构建者或 AI 产品策略师，这一天将为你提供构建可扩展、安全且面向未来的 MCP 解决方案所需的关键洞见。
