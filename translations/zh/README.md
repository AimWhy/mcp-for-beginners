# 🚀 初学者的模型上下文协议（MCP）课程

## **通过 C#、Java、JavaScript、Python 和 TypeScript 的实战代码示例学习 MCP**

## 🧠 模型上下文协议课程概览

**模型上下文协议（MCP）** 是一个前沿框架，旨在标准化 AI 模型与客户端应用之间的交互。这个开源课程提供了结构化的学习路径，配备实用的代码示例和真实案例，涵盖 C#、Java、JavaScript、TypeScript 和 Python 等主流编程语言。

无论你是 AI 开发者、系统架构师，还是软件工程师，这份指南都是你掌握 MCP 基础知识和实现策略的全面资源。

## 🔗 官方 MCP 资源

- 📘 [MCP 文档](https://modelcontextprotocol.io/) – 详细教程和用户指南  
- 📜 [MCP 规范](https://spec.modelcontextprotocol.io/) – 协议架构和技术参考  
- 🧑‍💻 [MCP GitHub 仓库](https://github.com/modelcontextprotocol) – 开源 SDK、工具和代码示例  

## 🧭 完整的 MCP 课程结构

| 章节 | 标题 | 说明 | 链接 |
|--|--|--|--|
| 00 | **MCP 介绍** | 介绍模型上下文协议及其在 AI 流水线中的重要性，包括什么是模型上下文协议、为何标准化至关重要，以及实际应用场景和优势 | [介绍](./00-Introduction/README.md) |
| 01 | **核心概念解析** | 深入探讨 MCP 的核心概念，包括客户端-服务器架构、关键协议组件和消息传递模式 | [核心概念](./01-CoreConcepts/README.md) |
| 02 | **MCP 安全性** | 识别基于 MCP 系统中的安全威胁，安全实现的技术和最佳实践 | [安全](./02-Security/README.md) |
| 03 | **MCP 入门** | 环境搭建与配置，创建基本的 MCP 服务器和客户端，将 MCP 集成到现有应用中 | [入门](./03-GettingStarted/README.md) |
| 3.1 | **第一个服务器** | 使用 MCP 协议搭建基础服务器，理解服务器与客户端的交互，并进行服务器测试 | [第一个服务器](./03-GettingStarted/01-first-server/README.md) |
| 3.2 | **第一个客户端** | 使用 MCP 协议搭建基础客户端，理解客户端与服务器的交互，并进行客户端测试 | [第一个客户端](./03-GettingStarted/02-client/README.md) |
| 3.3 | **带 LLM 的客户端** | 使用 MCP 协议搭建集成大型语言模型（LLM）的客户端 | [带 LLM 的客户端](./03-GettingStarted/03-llm-client/README.md) |
| 3.4 | **用 Visual Studio Code 连接服务器** | 配置 Visual Studio Code 以使用 MCP 协议连接服务器 | [用 Visual Studio Code 连接服务器](./03-GettingStarted/04-vscode/README.md) |
| 3.5 | **使用 SSE 创建服务器** | SSE 有助于将服务器暴露到互联网。本节将指导你使用 SSE 创建服务器 | [使用 SSE 创建服务器](./03-GettingStarted/05-sse-server/README.md) |
| 3.6 | **使用 AI 工具包** | AI 工具包是管理 AI 和 MCP 工作流程的好帮手 | [使用 AI 工具包](./03-GettingStarted/06-aitk/README.md) |
| 3.7 | **测试你的服务器** | 测试是开发过程的重要环节。本节将指导你使用多种工具进行测试 | [测试你的服务器](./03-GettingStarted/07-testing/README.md) |
| 3.8 | **部署你的服务器** | 如何从本地开发过渡到生产环境？本节将帮助你开发并部署服务器 | [部署你的服务器](./03-GettingStarted/08-deployment/README.md) |
| 04 | **实战应用** | 使用不同语言的 SDK，调试、测试和验证，设计可复用的提示模板和工作流 | [实战应用](./04-PracticalImplementation/README.md) |
| 05 | **MCP 高级话题** | 多模态 AI 工作流与扩展性，安全的扩展策略，MCP 在企业生态系统中的应用 | [高级话题](./05-AdvancedTopics/README.md) |
| 06 | **社区贡献** | 如何贡献代码和文档，通过 GitHub 协作，社区驱动的增强和反馈 | [社区贡献](./06-CommunityContributions/README.md) |
| 07 | **早期采用的经验分享** | 真实案例及成功经验，基于 MCP 的解决方案构建与部署，趋势和未来路线图 | [经验分享](./07-LessonsFromEarlyAdoption/README.md) |
| 08 | **MCP 最佳实践** | 性能调优与优化，设计容错 MCP 系统，测试和弹性策略 | [最佳实践](./08-BestPractices/README.md) |
| 09 | **MCP 案例研究** | 深入分析 MCP 解决方案架构、部署蓝图和集成技巧，带注释的图示和项目演示 | [案例研究](./09-CaseStudy/README.md) |

## 🎯 学习 MCP 的先决条件

为了充分利用本课程，您应具备：

- 基础的 C#、Java 或 Python 知识
- 了解客户端-服务器模型和 API
- （可选）熟悉机器学习的基本概念

## 🛠️ 如何高效使用本课程

本指南中的每节课包含：

1. 对 MCP 概念的清晰讲解  
2. 多种语言的实时代码示例  
3. 用于构建真实 MCP 应用的练习
