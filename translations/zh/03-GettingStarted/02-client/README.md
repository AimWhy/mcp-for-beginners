## 主要收获

本章关于客户端的主要收获包括：

- 客户端既可用于发现服务器功能，也可调用这些功能。
- 客户端可以在启动自身时启动服务器（如本章所示），也可以连接到已运行的服务器。
- 客户端是测试服务器功能的好方式，除了上一章提到的 Inspector 工具外。
