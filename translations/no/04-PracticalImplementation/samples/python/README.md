# Eksempel

Dette er et Python-eksempel for en MCP-server.

Denne modulen viser hvordan man implementerer en grunnleggende MCP-server som kan håndtere forespørsler om fullføring. Den gir en mock-implementering som simulerer interaksjon med forskjellige AI-modeller.

Slik ser registreringsprosessen for verktøyet ut:

```python
completion_tool = ToolDefinition(
    name="completion",
    description="Generate completions using AI models",
    parameters={
        "model": {
            "type": "string",
            "enum": self.models,
            "description": "The AI model to use for completion"
        },
        "prompt": {
            "type": "string",
            "description": "The prompt text to complete"
        },
        "temperature": {
            "type": "number",
            "description": "Sampling temperature (0.0 to 1.0)"
        },
        "max_tokens": {
            "type": "number",
            "description": "Maximum number of tokens to generate"
        }
    },
    required=["model", "prompt"]
)

# Register the tool with its handler
self.server.tools.register(completion_tool, self._handle_completion)
```

## Installere

Kjør følgende kommando:

```bash
pip install mcp
```

## Kjøre

```bash
python mcp_sample.py
```
