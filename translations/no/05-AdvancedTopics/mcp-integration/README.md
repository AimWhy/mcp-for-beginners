## Integrering av MCP med Azure ML

Å integrere MCP med Azure Machine Learning (ML) lar deg utnytte Azures kraftige ML-funksjoner samtidig som du beholder fleksibiliteten til MCP. Denne integrasjonen kan brukes til å kjøre ML-pipelines, registrere modeller som verktøy og administrere databehandlingsressurser.
