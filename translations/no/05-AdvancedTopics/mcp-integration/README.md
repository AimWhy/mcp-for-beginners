## Integrering av MCP med Azure ML

Å integrere MCP med Azure Machine Learning (ML) gjør det mulig å utnytte Azures kraftige ML-funksjoner samtidig som man beholder fleksibiliteten til MCP. Denne integrasjonen kan brukes til å kjøre ML-pipelines, registrere modeller som verktøy, og administrere databehandlingsressurser.
