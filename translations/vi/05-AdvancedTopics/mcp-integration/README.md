## Tích hợp MCP với Azure ML

Tích hợp MCP với Azure Machine Learning (ML) cho phép bạn tận dụng các khả năng ML mạnh mẽ của Azure đồng thời giữ được sự linh hoạt của MCP. Việc tích hợp này có thể được sử dụng để thực thi các pipeline ML, đăng ký các mô hình như công cụ, và quản lý tài nguyên tính toán.
