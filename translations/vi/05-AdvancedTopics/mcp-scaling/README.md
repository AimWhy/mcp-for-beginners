## Kiến trúc Phân tán

Kiến trúc phân tán bao gồm nhiều nút MCP làm việc cùng nhau để xử lý các yêu cầu, chia sẻ tài nguyên và cung cấp tính dự phòng. Cách tiếp cận này nâng cao khả năng mở rộng và chịu lỗi bằng cách cho phép các nút giao tiếp và phối hợp thông qua một hệ thống phân tán.

Hãy xem ví dụ về cách triển khai kiến trúc máy chủ MCP phân tán sử dụng Redis để điều phối.
