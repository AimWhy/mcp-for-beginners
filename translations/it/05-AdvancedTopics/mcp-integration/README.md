## Integrazione di MCP con Azure ML

Integrare MCP con Azure Machine Learning (ML) ti consente di sfruttare le potenti capacità di ML di Azure mantenendo la flessibilità di MCP. Questa integrazione può essere utilizzata per eseguire pipeline ML, registrare modelli come strumenti e gestire le risorse di calcolo.
