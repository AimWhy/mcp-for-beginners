# 🚀 Model Context Protocol (MCP) Curriculum for Beginners

## **Aprende MCP con ejemplos prácticos de código en C#, Java, JavaScript, Python y TypeScript**

## 🧠 Resumen del currículo del Model Context Protocol

El **Model Context Protocol (MCP)** es un marco innovador diseñado para estandarizar las interacciones entre modelos de IA y aplicaciones cliente. Este currículo de código abierto ofrece un camino de aprendizaje estructurado, con ejemplos prácticos de programación y casos de uso reales, en lenguajes populares como C#, Java, JavaScript, TypeScript y Python.

Ya seas desarrollador de IA, arquitecto de sistemas o ingeniero de software, esta guía es tu recurso completo para dominar los fundamentos y las estrategias de implementación de MCP.

## 🔗 Recursos oficiales de MCP

- 📘 [MCP Documentation](https://modelcontextprotocol.io/) – Tutoriales detallados y guías de usuario  
- 📜 [MCP Specification](https://spec.modelcontextprotocol.io/) – Arquitectura del protocolo y referencias técnicas  
- 🧑‍💻 [MCP GitHub Repository](https://github.com/modelcontextprotocol) – SDKs de código abierto, herramientas y ejemplos de código
