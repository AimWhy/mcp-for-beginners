# 🚀 Model Context Protocol (MCP) 初學者課程

## **透過 C#、Java、JavaScript、Python 和 TypeScript 實作範例學習 MCP**

## 🧠 Model Context Protocol 課程概覽

**Model Context Protocol (MCP)** 是一個前沿框架，旨在標準化 AI 模型與客戶端應用程式之間的互動。這套開源課程提供結構化的學習路徑，包含實用的程式碼範例與真實案例，涵蓋熱門程式語言如 C#、Java、JavaScript、TypeScript 及 Python。

無論你是 AI 開發者、系統架構師或軟體工程師，本指南都是你掌握 MCP 基礎與實作策略的全面資源。

## 🔗 官方 MCP 資源

- 📘 [MCP 文件](https://modelcontextprotocol.io/) – 詳盡的教學與使用指南  
- 📜 [MCP 規範](https://spec.modelcontextprotocol.io/) – 協議架構與技術參考  
- 🧑‍💻 [MCP GitHub 倉庫](https://github.com/modelcontextprotocol) – 開源 SDK、工具與程式碼範例  

## 加入我們，參加 2025 年 7 月 29-30 日 MCP 開發日

準備好迎接兩天深度技術分享、社群交流與實作學習的 MCP 開發日，這是一場專注於 Model Context Protocol（MCP）的線上活動——這個新興標準正連結 AI 模型與其依賴的工具。

➡️ [報名 MCP 開發日](https://developer.microsoft.com/en-us/reactor/series/S-1563/)

你可以在活動頁面 https://aka.ms/mcpdevdays 註冊觀看 MCP 開發日，屆時可透過 YouTube 或 Twitch 直播參與。所有內容會錄製並於活動後上傳至 Microsoft Developer YouTube 頻道，示範程式碼也會同步在 GitHub 上提供。

### 活動詳情
- 日期：7 月 29 日（第一天）及 7 月 30 日（第二天）
- 時間：每日上午 9:00（太平洋標準時間）
- 地點：線上，隨時隨地皆可參加！

#### 第一天：MCP 生產力、開發工具與社群

專注於協助開發者將 MCP 融入工作流程，並慶祝精彩的 MCP 社群。我們將邀請社群成員與合作夥伴如 Arcade、Block、Okta 和 Neon，一同分享他們如何與微軟合作打造開放且可擴展的 MCP 生態系。包含 VS Code、Visual Studio、GitHub Copilot 及熱門社群工具的真實示範，實務且以情境為導向的開發流程，社群主導的議題與洞見。

無論你是 MCP 新手還是已經在使用 MCP，第一天都會帶來啟發與可行的策略。

#### 第二天：自信打造 MCP 伺服器

專為 MCP 建置者設計。我們將深入探討 MCP 伺服器的實作策略與最佳實務，以及如何將 MCP 整合到你的 AI 工作流程中。

### 主題包括：

- 建立 MCP 伺服器並整合至代理體驗
- 以提示驅動的開發
- 安全最佳實務
- 使用 Functions、ACA 及 API 管理等建構模組
- 註冊表對齊與工具（第一方 + 第三方）

如果你是開發者、工具建置者或 AI 產品策略師，這一天充滿你需要的見解，幫助你打造可擴展、安全且面向未來的 MCP 解決方案。
