# Ejemplo

Este es un ejemplo en Python para un servidor MCP.

Este módulo demuestra cómo implementar un servidor MCP básico que puede manejar
solicitudes de completado. Proporciona una implementación simulada que imita
la interacción con varios modelos de IA.

Así es como se ve el proceso de registro de herramientas:

```python
completion_tool = ToolDefinition(
    name="completion",
    description="Generate completions using AI models",
    parameters={
        "model": {
            "type": "string",
            "enum": self.models,
            "description": "The AI model to use for completion"
        },
        "prompt": {
            "type": "string",
            "description": "The prompt text to complete"
        },
        "temperature": {
            "type": "number",
            "description": "Sampling temperature (0.0 to 1.0)"
        },
        "max_tokens": {
            "type": "number",
            "description": "Maximum number of tokens to generate"
        }
    },
    required=["model", "prompt"]
)

# Register the tool with its handler
self.server.tools.register(completion_tool, self._handle_completion)
```

## Instalación

Ejecute el siguiente comando:

```bash
pip install mcp
```

## Ejecutar

```bash
python mcp_sample.py
```
