## Integrácia MCP s Azure ML

Integrácia MCP s Azure Machine Learning (ML) vám umožňuje využiť silné ML schopnosti Azure a zároveň zachovať flexibilitu MCP. Táto integrácia sa dá použiť na spúšťanie ML pipeline, registráciu modelov ako nástrojov a správu výpočtových zdrojov.
