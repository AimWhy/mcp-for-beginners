## Integrácia MCP s Azure ML

Integrácia MCP s Azure Machine Learning (ML) vám umožňuje využiť výkonné ML možnosti Azure pri zachovaní flexibility MCP. Táto integrácia môže slúžiť na spúšťanie ML pipeline, registráciu modelov ako nástrojov a správu výpočtových zdrojov.
