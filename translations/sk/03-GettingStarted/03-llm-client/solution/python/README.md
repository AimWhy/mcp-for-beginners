# Spustenie tejto ukážky

Odporúča sa nainštalovať `uv`, ale nie je to nutné, pozri [inštrukcie](https://docs.astral.sh/uv/#highlights)

## -0- Vytvorte virtuálne prostredie

```bash
python -m venv venv
```

## -1- Aktivujte virtuálne prostredie

```bash
venv\Scrips\activate
```

## -2- Nainštalujte závislosti

```bash
pip install "mcp[cli]"
pip install openai
pip install azure-ai-inference
```

## -3- Spustite ukážku

```bash
python client.py
```
