## Integracja MCP z Azure ML

Integracja MCP z Azure Machine Learning (ML) pozwala wykorzystać potężne możliwości ML oferowane przez Azure, zachowując jednocześnie elastyczność MCP. Dzięki temu połączeniu możesz uruchamiać potoki ML, rejestrować modele jako narzędzia oraz zarządzać zasobami obliczeniowymi.
