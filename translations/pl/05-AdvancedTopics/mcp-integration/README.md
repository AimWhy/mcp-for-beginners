## Integracja MCP z Azure ML

Integracja MCP z Azure Machine Learning (ML) pozwala wykorzystać potężne możliwości ML oferowane przez Azure, jednocześnie zachowując elastyczność MCP. Ta integracja umożliwia wykonywanie pipeline'ów ML, rejestrowanie modeli jako narzędzi oraz zarządzanie zasobami obliczeniowymi.
