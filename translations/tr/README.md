# 🚀 Başlangıç Seviyesi için Model Context Protocol (MCP) Müfredatı

## **C#, Java, JavaScript, Python ve TypeScript ile MCP'yi Uygulamalı Kod Örnekleriyle Öğrenin**

## 🧠 Model Context Protocol Müfredatına Genel Bakış

**Model Context Protocol (MCP)**, yapay zeka modelleri ile istemci uygulamaları arasındaki etkileşimleri standartlaştırmak için tasarlanmış yenilikçi bir çerçevedir. Bu açık kaynaklı müfredat, C#, Java, JavaScript, TypeScript ve Python gibi popüler programlama dillerinde pratik kod örnekleri ve gerçek dünya kullanım senaryoları içeren yapılandırılmış bir öğrenme yolu sunar.

İster bir yapay zeka geliştiricisi, sistem mimarı ya da yazılım mühendisi olun, bu rehber MCP temellerini ve uygulama stratejilerini öğrenmeniz için kapsamlı bir kaynaktır.

## 🔗 Resmi MCP Kaynakları

- 📘 [MCP Dokümantasyonu](https://modelcontextprotocol.io/) – Detaylı eğitimler ve kullanıcı kılavuzları  
- 📜 [MCP Spesifikasyonu](https://spec.modelcontextprotocol.io/) – Protokol mimarisi ve teknik referanslar  
- 🧑‍💻 [MCP GitHub Deposu](https://github.com/modelcontextprotocol) – Açık kaynak SDK’lar, araçlar ve kod örnekleri  

## 29-30 Temmuz 2025’te MCP Dev Days’e Katılın

Model Context Protocol (MCP) için düzenlenen, AI modelleri ile kullandıkları araçları birleştiren yeni standart üzerine odaklanan MCP Dev Days etkinliğinde, iki gün boyunca derin teknik bilgiler, topluluk bağlantıları ve uygulamalı öğrenme sizi bekliyor.

➡️ [MCP Dev Days için Kayıt Olun](https://developer.microsoft.com/en-us/reactor/series/S-1563/)

MCP Dev Days’i etkinlik sayfamızdan kayıt olarak izleyebilirsiniz: https://aka.ms/mcpdevdays. Buradan YouTube veya Twitch üzerinden canlı yayına katılabilirsiniz. Tüm içerikler kaydedilecek ve daha sonra Microsoft Developer YouTube kanalında erişime açılacak. Demo kaynak kodları da GitHub’da paylaşılacak.

### Etkinlik Detayları
- Tarihler: 29 Temmuz (1. Gün) & 30 Temmuz (2. Gün)
- Saat: Her gün 9:00 AM PST
- Nerede: Çevrimiçi – her yerden katılabilirsiniz!

#### 1. Gün: MCP Verimliliği, Geliştirme Araçları ve Topluluk

Geliştiricilerin MCP’yi iş akışlarına dahil etmelerini güçlendirmeye ve harika MCP topluluğunu kutlamaya odaklanır. Microsoft ile iş birliği içinde açık ve genişletilebilir bir MCP ekosistemini şekillendiren Arcade, Block, Okta ve Neon gibi topluluk üyeleri ve ortaklarıyla birlikte olacağız.  
Gerçek dünya demoları VS Code, Visual Studio, GitHub Copilot ve popüler topluluk araçları üzerinden  
Pratik, bağlama dayalı geliştirme iş akışları  
Topluluk liderliğinde oturumlar ve içgörüler  
MCP ile yeni tanışıyor ya da zaten kullanıyorsanız, 1. Gün ilham verici ve uygulanabilir çıkarımlarla sizi hazırlar.

#### 2. Gün: MCP Sunucularını Güvenle İnşa Edin

MCP geliştiricileri için. MCP sunucuları oluşturma ve MCP’yi yapay zeka iş akışlarınıza entegre etme konusunda uygulama stratejileri ve en iyi uygulamalar derinlemesine ele alınacak.

### Konular:

- MCP Sunucuları oluşturmak ve bunları ajan deneyimlerine entegre etmek  
- Komut tabanlı geliştirme  
- Güvenlik en iyi uygulamaları  
- Fonksiyonlar, ACA ve API Yönetimi gibi yapı taşlarını kullanmak  
- Kayıt uyumu ve araçlar (1P + 3P)

Eğer geliştirici, araç yapımcısı ya da yapay zeka ürün stratejistiyseniz, bu gün ölçeklenebilir, güvenli ve geleceğe hazır MCP çözümleri inşa etmek için ihtiyaç duyduğunuz içgörülerle dolu.
