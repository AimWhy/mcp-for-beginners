## Integracija MCP z Azure ML

Integracija MCP z Azure Machine Learning (ML) vam omogoča, da izkoristite zmogljive ML zmogljivosti Azure, hkrati pa ohranite prilagodljivost MCP. Ta integracija se lahko uporablja za izvajanje ML cevovodov, registracijo modelov kot orodij in upravljanje računalniških virov.
