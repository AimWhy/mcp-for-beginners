## ตัวอย่างมัลติ-โหมด: การประมวลผลเสียง

การประมวลผลเสียงเป็นโหมดที่ใช้บ่อยในแอปพลิเคชันมัลติ-โหมด ด้านล่างเป็นตัวอย่างการใช้งานเครื่องมือถอดเสียงที่สามารถจัดการไฟล์เสียงและส่งคืนข้อความถอดเสียงได้

**ข้อจำกัดความรับผิดชอบ**:  
เอกสารฉบับนี้ได้รับการแปลโดยใช้บริการแปลภาษาอัตโนมัติ [Co-op Translator](https://github.com/Azure/co-op-translator) แม้เราจะพยายามให้ความถูกต้องสูงสุด โปรดทราบว่าการแปลโดยอัตโนมัติอาจมีข้อผิดพลาดหรือความคลาดเคลื่อนได้ เอกสารต้นฉบับในภาษาต้นฉบับถือเป็นแหล่งข้อมูลที่น่าเชื่อถือที่สุด สำหรับข้อมูลที่สำคัญ ขอแนะนำให้ใช้บริการแปลโดยผู้เชี่ยวชาญด้านภาษามนุษย์ เราจะไม่รับผิดชอบต่อความเข้าใจผิดหรือการตีความผิดที่เกิดขึ้นจากการใช้การแปลนี้
