## ตัวอย่างมัลติ-โหมด: การประมวลผลเสียง

การประมวลผลเสียงเป็นโหมดหนึ่งที่ใช้บ่อยในแอปพลิเคชันมัลติ-โหมด ด้านล่างนี้เป็นตัวอย่างการสร้างเครื่องมือถอดเสียงที่สามารถจัดการไฟล์เสียงและส่งคืนข้อความถอดเสียงได้

### ตัวอย่างมัลติ-โหมด: การสร้างการตอบสนองแบบมัลติ-โหมด

## ต่อไปคืออะไร

- [5.3 Oauth 2](../mcp-oauth2-demo/README.md)

**ข้อจำกัดความรับผิดชอบ**:  
เอกสารนี้ได้รับการแปลโดยใช้บริการแปลภาษาอัตโนมัติ [Co-op Translator](https://github.com/Azure/co-op-translator) แม้เราจะพยายามให้ความถูกต้องสูงสุด แต่โปรดทราบว่าการแปลอัตโนมัติอาจมีข้อผิดพลาดหรือความไม่ถูกต้อง เอกสารต้นฉบับในภาษาต้นทางถือเป็นแหล่งข้อมูลที่เชื่อถือได้ สำหรับข้อมูลที่สำคัญ ขอแนะนำให้ใช้บริการแปลโดยผู้เชี่ยวชาญมนุษย์ เราไม่รับผิดชอบต่อความเข้าใจผิดหรือการตีความผิดใด ๆ ที่เกิดจากการใช้การแปลนี้
