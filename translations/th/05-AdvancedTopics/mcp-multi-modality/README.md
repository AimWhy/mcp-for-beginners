## ตัวอย่างมัลติ-โมดอล: การประมวลผลเสียง

การประมวลผลเสียงเป็นอีกหนึ่งรูปแบบที่พบบ่อยในแอปพลิเคชันมัลติ-โมดอล ด้านล่างนี้เป็นตัวอย่างการใช้งานเครื่องมือถอดเสียงที่สามารถจัดการกับไฟล์เสียงและส่งคืนข้อความถอดเสียงได้

### ตัวอย่างมัลติ-โมดอล: การสร้างการตอบสนองแบบมัลติ-โมดอล

## ต่อไปคืออะไร

- [Oauth 2](../mcp-oauth2-demo/README.md)

**ข้อจำกัดความรับผิดชอบ**:  
เอกสารนี้ได้รับการแปลโดยใช้บริการแปลภาษาอัตโนมัติ [Co-op Translator](https://github.com/Azure/co-op-translator) แม้เราจะพยายามให้ความถูกต้องสูงสุด แต่โปรดทราบว่าการแปลอัตโนมัติอาจมีข้อผิดพลาดหรือความคลาดเคลื่อน เอกสารต้นฉบับในภาษาต้นฉบับควรถูกพิจารณาเป็นแหล่งข้อมูลที่เชื่อถือได้ สำหรับข้อมูลที่สำคัญ แนะนำให้ใช้บริการแปลโดยผู้เชี่ยวชาญด้านภาษามนุษย์ เราจะไม่รับผิดชอบต่อความเข้าใจผิดหรือการตีความผิดใด ๆ ที่เกิดขึ้นจากการใช้การแปลนี้
