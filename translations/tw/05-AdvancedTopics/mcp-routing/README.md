## MCP中的抽樣與路由架構

抽樣是Model Context Protocol (MCP)中的關鍵組件，它允許有效率地處理與路由請求。透過分析進來的請求，根據內容類型、使用者情境和系統負載等各種條件，判斷最適合處理的模型或服務。

抽樣與路由可以結合，打造出一個強健的架構，優化資源使用並確保高可用性。抽樣過程用來分類請求，而路由則將請求導向適當的模型或服務。

以下圖示說明抽樣與路由如何在完整的MCP架構中協同運作：
