# 🚀 新手必備的 Model Context Protocol (MCP) 課程終極指南

## **用 C#、Java、JavaScript、Python 和 TypeScript 實作 MCP 範例程式碼**

## 🧠 Model Context Protocol 課程概覽

**Model Context Protocol (MCP)** 是一套先進的框架，旨在標準化 AI 模型與客戶端應用程式間的互動。這份開源課程提供結構化的學習路徑，搭配實務程式範例和真實案例，涵蓋 C#、Java、JavaScript、TypeScript 及 Python 等熱門程式語言。

不論你是 AI 開發者、系統架構師或軟體工程師，本指南都是你掌握 MCP 基礎與實作策略的完整資源。

## 🔗 官方 MCP 資源

- 📘 [MCP Documentation](https://modelcontextprotocol.io/) – 詳細教學與使用指南  
- 📜 [MCP Specification](https://spec.modelcontextprotocol.io/) – 協議架構與技術參考  
- 🧑‍💻 [MCP GitHub Repository](https://github.com/modelcontextprotocol) – 開源 SDK、工具與程式範例

|--|--|--|

| **02 - MCP 的安全性** | 辨識 MCP 系統中的安全威脅，並介紹保障實作的技術與最佳做法 | [Security](/02-Security/readme.md) |
| **03 - MCP 快速上手** | 環境設定與配置，建立基本的 MCP 伺服器與客戶端，並將 MCP 整合到現有應用程式中 | [Getting Started](./03-GettingStarted/README.md) |
| **3.1 - 第一個伺服器** | 使用 MCP 協議架設基本伺服器，了解伺服器與客戶端的互動，並進行伺服器測試 | [First Server](./03-GettingStarted/01-first-server/README.md) |

| **3.8 - 部署你的伺服器** | 從本地開發到正式上線，這章節會協助你開發並部署伺服器 | [Deploy your server](./03-GettingStarted/08-deployment/README.md) |
| **04 - 實務應用** | 跨語言使用 SDK，除錯、測試與驗證，打造可重用的提示模板與工作流程 | [Practical Implementation](./04-PracticalImplementation/README.md) |
| **05 - MCP 進階主題** | 多模態 AI 工作流程與擴充性、安全擴展策略、企業生態系中的 MCP | [Advanced Topics](./05-AdvancedTopics/README.md) |

| **07 - 早期採用的洞見** | 真實案例分享與成效，MCP 解決方案的建置與部署，趨勢與未來路線圖 | [Insights](./07-LessonsFromEarlyAdoption/README.md) |
| **08 - MCP 最佳實務** | 效能調校與優化，設計容錯的 MCP 系統，測試與韌性策略 | [Best Practices](./08-BestPractices/README.md) |

## 🎯 學習 MCP 的先備條件

要充分利用這個課程，你應該具備：

- 基本的 C#、Java 或 Python 知識
- 了解客戶端-伺服器模型及 API
- （選擇性）熟悉機器學習概念
