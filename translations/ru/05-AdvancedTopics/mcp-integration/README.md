## Интеграция MCP с Azure ML

Интеграция MCP с Azure Machine Learning (ML) позволяет использовать мощные возможности Azure ML, сохраняя при этом гибкость MCP. Такая интеграция может использоваться для выполнения ML-пайплайнов, регистрации моделей в качестве инструментов и управления вычислительными ресурсами.
