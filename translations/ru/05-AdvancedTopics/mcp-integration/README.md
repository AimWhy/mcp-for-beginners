## Интеграция MCP с Azure ML

Интеграция MCP с Azure Machine Learning (ML) позволяет использовать мощные возможности машинного обучения Azure, сохраняя при этом гибкость MCP. Эта интеграция может использоваться для выполнения ML-пайплайнов, регистрации моделей в качестве инструментов и управления вычислительными ресурсами.
