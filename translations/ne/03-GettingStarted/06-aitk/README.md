# Visual Studio Code को लागि AI Toolkit विस्तारबाट सर्वर उपभोग गर्ने

जब तपाईं एक AI एजेन्ट निर्माण गर्दै हुनुहुन्छ, यो केवल स्मार्ट प्रतिक्रिया उत्पन्न गर्ने बारे मात्र होइन; यो पनि तपाईंको एजेन्टलाई क्रियाकलाप गर्न सक्ने क्षमता दिने बारे हो। यहीँ Model Context Protocol (MCP) आउँछ। MCP ले एजेन्टहरूलाई बाह्य उपकरण र सेवाहरूलाई सजिलो तरिकाले पहुँच गर्न मद्दत गर्दछ। यो तपाईंको एजेन्टलाई वास्तवमा प्रयोग गर्न सक्ने टूलबक्समा जोड्ने जस्तै हो।

मानौं तपाईंले आफ्नो एजेन्टलाई क्यालकुलेटर MCP सर्वरमा जडान गर्नुभयो। अचानक, तपाईंको एजेन्टले गणितीय कार्यहरू गर्न सक्छ केवल "47 गुणा 89 के हो?" जस्तो प्रॉम्प्ट प्राप्त गरेर—कुनै हार्डकोडिङ तार्किकता वा कस्टम API बनाउन आवश्यक छैन।

## अवलोकन

यस पाठले Visual Studio Code मा [AI Toolkit](https://aka.ms/AIToolkit) विस्तारको साथ क्यालकुलेटर MCP सर्वरलाई कसरी एजेन्टमा जडान गर्ने कभर गर्दछ, जसले तपाईंको एजेन्टलाई प्राकृतिक भाषाको माध्यमबाट थप, घट, गुणा, र भाग जस्ता गणितीय कार्यहरू गर्न सक्षम बनाउँछ।

AI Toolkit Visual Studio Code को लागि एक शक्तिशाली विस्तार हो जसले एजेन्ट विकासलाई सरल बनाउँछ। AI इन्जिनियरहरूले सजिलै AI अनुप्रयोगहरू निर्माण गर्न र परीक्षण गर्न सक्दछन्—स्थानीय रूपमा वा क्लाउडमा। विस्तारले आज उपलब्ध प्रमुख जेनरेटिव मोडेलहरूको समर्थन गर्दछ।

*नोट*: AI Toolkit हाल Python र TypeScript लाई समर्थन गर्दछ।

## सिकाइ उद्देश्यहरू

यस पाठको अन्त्यमा, तपाईं सक्षम हुनुहुनेछ:

- AI Toolkit मार्फत MCP सर्वर उपभोग गर्न।
- MCP सर्वरद्वारा प्रदान गरिएका उपकरणहरू पत्ता लगाउन र प्रयोग गर्न सक्षम बनाउन एजेन्ट कन्फिगरेसनलाई कन्फिगर गर्न।
- प्राकृतिक भाषाको माध्यमबाट MCP उपकरणहरू प्रयोग गर्न।

## दृष्टिकोण

हामीलाई उच्च स्तरमा यसलाई कसरी दृष्टिकोण गर्न आवश्यक छ:

- एक एजेन्ट सिर्जना गर्नुहोस् र यसको प्रणाली प्रॉम्प्ट परिभाषित गर्नुहोस्।
- क्यालकुलेटर उपकरणहरूसँग MCP सर्वर सिर्जना गर्नुहोस्।
- Agent Builder लाई MCP सर्वरमा जडान गर्नुहोस्।
- प्राकृतिक भाषाको माध्यमबाट एजेन्टको उपकरण आह्वान परीक्षण गर्नुहोस्।

ठीक छ, अब हामी प्रवाहलाई बुझ्यौं, आउनुहोस् एक AI एजेन्टलाई बाह्य उपकरणहरू मार्फत MCP को उपयोग गर्न कन्फिगर गरौं, यसको क्षमता बढाउँदै!

## अभ्यास: सर्वर उपभोग गर्ने

यस अभ्यासमा, तपाईं AI Toolkit प्रयोग गरेर Visual Studio Code भित्र एक MCP सर्वरबाट उपकरणहरूसँग एक AI एजेन्ट निर्माण, चलाउन, र सुधार गर्नेछ।

### -0- पूर्व चरण, OpenAI GPT-4o मोडेललाई My Models मा थप्नुहोस्

अभ्यासले **GPT-4o** मोडेललाई उपयोग गर्दछ। एजेन्ट सिर्जना गर्नु अघि मोडेललाई **My Models** मा थप्नुपर्छ।

1. **Activity Bar** बाट **AI Toolkit** विस्तार खोल्नुहोस्।
1. **Catalog** सेक्सनमा, **Models** चयन गर्नुहोस् **Model Catalog** खोल्न। **Models** चयन गर्दा **Model Catalog** नयाँ एडिटर ट्याबमा खुल्छ।
1. **Model Catalog** खोज बारमा **OpenAI GPT-4o** प्रविष्ट गर्नुहोस्।
1. मोडेललाई तपाईंको **My Models** सूचीमा थप्न **+ Add** क्लिक गर्नुहोस्। सुनिश्चित गर्नुहोस् कि तपाईंले **Hosted by GitHub** मोडेल चयन गर्नुभएको छ।
1. **Activity Bar** मा, **OpenAI GPT-4o** मोडेल सूचीमा देखिन्छ भनेर पुष्टि गर्नुहोस्।

### -1- एक एजेन्ट सिर्जना गर्नुहोस्

**Agent (Prompt) Builder** ले तपाईंलाई आफ्नो AI-संचालित एजेन्टहरू सिर्जना र अनुकूलित गर्न सक्षम बनाउँछ। यस सेक्सनमा, तपाईं एक नयाँ एजेन्ट सिर्जना गर्नुहुनेछ र वार्तालापलाई शक्ति प्रदान गर्न मोडेललाई असाइन गर्नुहुनेछ।

1. **Activity Bar** बाट **AI Toolkit** विस्तार खोल्नुहोस्।
1. **Tools** सेक्सनमा, **Agent (Prompt) Builder** चयन गर्नुहोस्। **Agent (Prompt) Builder** चयन गर्दा **Agent (Prompt) Builder** नयाँ एडिटर ट्याबमा खुल्छ।
1. **+ New Builder** बटन क्लिक गर्नुहोस्। विस्तारले **Command Palette** मार्फत सेटअप विजार्ड सुरु गर्नेछ।
1. **Calculator Agent** नाम प्रविष्ट गर्नुहोस् र **Enter** थिच्नुहोस्।
1. **Agent (Prompt) Builder** मा, **Model** क्षेत्रको लागि, **OpenAI GPT-4o (via GitHub)** मोडेल चयन गर्नुहोस्।

### -2- एजेन्टको लागि प्रणाली प्रॉम्प्ट सिर्जना गर्नुहोस्

एजेन्टको प्रणाली प्रॉम्प्टलाई परिभाषित गर्न समय आएको छ—यसको व्यक्तित्व र उद्देश्यलाई मार्गदर्शन गर्दै। यस सेक्सनमा, तपाईं **Generate system prompt** सुविधा प्रयोग गरेर एजेन्टको इच्छित व्यवहार वर्णन गर्नुहुनेछ—यस अवस्थामा, एक क्यालकुलेटर एजेन्ट—र मोडेलले तपाईंको लागि प्रणाली प्रॉम्प्ट लेख्नेछ।

1. **Prompts** सेक्सनको लागि, **Generate system prompt** बटन क्लिक गर्नुहोस्। यो बटन प्रॉम्प्ट बिल्डरमा खुल्छ जसले एजेन्टको लागि प्रणाली प्रॉम्प्ट उत्पन्न गर्न AI प्रयोग गर्दछ।
1. **Generate a prompt** विन्डोमा, निम्न प्रविष्ट गर्नुहोस्: `You are a helpful and efficient math assistant. When given a problem involving basic arithmetic, you respond with the correct result.`
1. **Generate** बटन क्लिक गर्नुहोस्। प्रणाली प्रॉम्प्ट उत्पन्न भइरहेको पुष्टि गर्दै तल-दायाँ कुनामा सूचना देखा पर्नेछ। प्रॉम्प्ट उत्पन्न समाप्त भएपछि, प्रॉम्प्ट **Agent (Prompt) Builder** को **System prompt** क्षेत्रमा देखा पर्नेछ।
1. **System prompt** समीक्षा गर्नुहोस् र आवश्यक भएमा संशोधन गर्नुहोस्।

### -3- MCP सर्वर सिर्जना गर्नुहोस्

अब तपाईंले आफ्नो एजेन्टको प्रणाली प्रॉम्प्टलाई परिभाषित गर्नुभयो—यसको व्यवहार र प्रतिक्रियाहरूलाई मार्गदर्शन गर्दै—यसलाई व्यावहारिक क्षमताहरू प्रदान गर्ने समय आएको छ। यस सेक्सनमा, तपाईं एक क्यालकुलेटर MCP सर्वर सिर्जना गर्नुहुनेछ जसले जोड, घट, गुणा, र भाग गणनाहरू कार्यान्वयन गर्न उपकरणहरू प्रदान गर्नेछ। यस सर्वरले तपाईंको एजेन्टलाई प्राकृतिक भाषाको प्रॉम्प्टहरूमा वास्तविक-समय गणितीय कार्यहरू गर्न सक्षम बनाउनेछ।

AI Toolkit आफ्नो MCP सर्वर सिर्जना गर्न सजिलोको लागि टेम्पलेटहरूसँग सुसज्जित छ। हामी क्यालकुलेटर MCP सर्वर सिर्जना गर्न Python टेम्पलेट प्रयोग गर्नेछौं।

*नोट*: AI Toolkit हाल Python र TypeScript लाई समर्थन गर्दछ।

1. **Agent (Prompt) Builder** को **Tools** सेक्सनमा, **+ MCP Server** बटन क्लिक गर्नुहोस्। विस्तारले **Command Palette** मार्फत सेटअप विजार्ड सुरु गर्नेछ।
1. **+ Add Server** चयन गर्नुहोस्।
1. **Create a New MCP Server** चयन गर्नुहोस्।
1. **python-weather** टेम्पलेट चयन गर्नुहोस्।
1. MCP सर्वर टेम्पलेट बचत गर्न **Default folder** चयन गर्नुहोस्।
1. सर्वरको लागि निम्न नाम प्रविष्ट गर्नुहोस्: **Calculator**
1. नयाँ Visual Studio Code विन्डो खुल्नेछ। **Yes, I trust the authors** चयन गर्नुहोस्।
1. टर्मिनल प्रयोग गरेर (**Terminal** > **New Terminal**), भर्चुअल वातावरण सिर्जना गर्नुहोस्: `python -m venv .venv`
1. टर्मिनल प्रयोग गरेर, भर्चुअल वातावरण सक्रिय गर्नुहोस्:
    1. Windows - `.venv\Scripts\activate`
    1. macOS/Linux - `source venv/bin/activate`
1. टर्मिनल प्रयोग गरेर, निर्भरता स्थापना गर्नुहोस्: `pip install -e .[dev]`
1. **Activity Bar** को **Explorer** दृश्यमा, **src** निर्देशिका विस्तार गर्नुहोस् र **server.py** चयन गर्नुहोस् फाइललाई एडिटरमा खोल्न।
1. **server.py** फाइलको कोडलाई निम्नसँग बदल्नुहोस् र बचत गर्नुहोस्:

    ```python
    """
    Sample MCP Calculator Server implementation in Python.

    
    This module demonstrates how to create a simple MCP server with calculator tools
    that can perform basic arithmetic operations (add, subtract, multiply, divide).
    """
    
    from mcp.server.fastmcp import FastMCP
    
    server = FastMCP("calculator")
    
    @server.tool()
    def add(a: float, b: float) -> float:
        """Add two numbers together and return the result."""
        return a + b
    
    @server.tool()
    def subtract(a: float, b: float) -> float:
        """Subtract b from a and return the result."""
        return a - b
    
    @server.tool()
    def multiply(a: float, b: float) -> float:
        """Multiply two numbers together and return the result."""
        return a * b
    
    @server.tool()
    def divide(a: float, b: float) -> float:
        """
        Divide a by b and return the result.
        
        Raises:
            ValueError: If b is zero
        """
        if b == 0:
            raise ValueError("Cannot divide by zero")
        return a / b
    ```

### -4- क्यालकुलेटर MCP सर्वरको साथ एजेन्ट चलाउनुहोस्

अब तपाईंको एजेन्टसँग उपकरणहरू छन्, अब तिनीहरूलाई प्रयोग गर्ने समय आएको छ! यस सेक्सनमा, तपाईं एजेन्टलाई प्रॉम्प्टहरू पेश गर्नुहुनेछ एजेन्टले क्यालकुलेटर MCP सर्वरबाट उपयुक्त उपकरण प्रयोग गर्छ कि छैन परीक्षण र मान्य गर्न।

तपाईं आफ्नो स्थानीय विकास मेशिनमा **Agent Builder** मार्फत क्यालकुलेटर MCP सर्वर चलाउनुहुनेछ जसको रूपमा MCP क्लाइन्ट।

1. `F5` to start debugging the MCP server. The **Agent (Prompt) Builder** will open in a new editor tab. The status of the server is visible in the terminal.
1. In the **User prompt** field of the **Agent (Prompt) Builder**, enter the following prompt: `म 3 वस्तुहरू खरीद गरें जसको मूल्य प्रत्येक $25 थियो, र त्यसपछि $20 छुट प्रयोग गरें। मैले कति तिरेँ?`
1. Click the **Run** button to generate the agent's response.
1. Review the agent output. The model should conclude that you paid **$55**.
1. Here's a breakdown of what should occur:
    - The agent selects the **multiply** and **substract** tools to aid in the calculation.
    - The respective `a` and `b` values are assigned for the **multiply** tool.
    - The respective `a` and `b` मूल्यहरू **subtract** उपकरणको लागि असाइन गरिन्छ।
    - प्रत्येक उपकरणको प्रतिक्रिया सम्बन्धित **Tool Response** मा प्रदान गरिएको छ।
    - मोडेलको अन्तिम आउटपुट अन्तिम **Model Response** मा प्रदान गरिएको छ।
1. एजेन्टलाई थप परीक्षण गर्न थप प्रॉम्प्टहरू पेश गर्नुहोस्। तपाईं **User prompt** क्षेत्रमा रहेको प्रॉम्प्टलाई क्लिक गरेर र प्रतिस्थापन गरेर संशोधन गर्न सक्नुहुन्छ।
1. एक पटक तपाईं एजेन्टको परीक्षण समाप्त गरेपछि, सर्वरलाई **terminal** मार्फत **CTRL/CMD+C** प्रविष्ट गरेर रोक्न सक्नुहुन्छ।

## असाइनमेन्ट

तपाईंको **server.py** फाइलमा अतिरिक्त उपकरण प्रविष्टि थप्ने प्रयास गर्नुहोस् (उदाहरण: एक नम्बरको वर्गमूल फर्काउने)। तपाईंको नयाँ उपकरण (वा विद्यमान उपकरणहरू) प्रयोग गर्न एजेन्टलाई आवश्यक पर्ने थप प्रॉम्प्टहरू पेश गर्नुहोस्। नयाँ उपकरणहरू लोड गर्न सर्वरलाई पुनः सुरु गर्न निश्चित गर्नुहोस्।

यो दस्तावेज AI अनुवाद सेवा [Co-op Translator](https://github.com/Azure/co-op-translator) प्रयोग गरी अनुवाद गरिएको हो। हामी शुद्धताका लागि प्रयास गर्छौं, तर कृपया सचेत रहनुहोस् कि स्वचालित अनुवादहरूमा त्रुटिहरू वा अशुद्धताहरू हुन सक्छन्। यसको मूल भाषामा रहेको मूल दस्तावेजलाई प्राधिकृत स्रोतको रूपमा मान्नुपर्छ। महत्वपूर्ण जानकारीको लागि, व्यावसायिक मानव अनुवाद सिफारिस गरिन्छ। यस अनुवादको प्रयोगबाट उत्पन्न हुने कुनै पनि गलतफहमी वा गलत व्याख्याका लागि हामी उत्तरदायी हुनेछैनौं।
