# üöÄ Model Context Protocol (MCP) Curriculum para sa mga Baguhan

## **Matutunan ang MCP gamit ang Mga Praktikal na Halimbawa ng Code sa C#, Java, JavaScript, Python, at TypeScript**

## üß† Pangkalahatang-ideya ng Model Context Protocol Curriculum

Ang **Model Context Protocol (MCP)** ay isang makabagong framework na idinisenyo upang gawing standard ang ugnayan sa pagitan ng mga AI model at mga client application. Ang open-source na curriculum na ito ay nag-aalok ng isang organisadong landas ng pag-aaral, kumpleto sa mga praktikal na halimbawa ng coding at totoong gamit, sa mga sikat na programming language tulad ng C#, Java, JavaScript, TypeScript, at Python.

Kung ikaw man ay isang AI developer, system architect, o software engineer, ang gabay na ito ang iyong komprehensibong sanggunian para sa pag-master ng mga pundasyon at estratehiya sa implementasyon ng MCP.

## üîó Opisyal na Mga Sanggunian ng MCP

- üìò [MCP Documentation](https://modelcontextprotocol.io/) ‚Äì Detalyadong mga tutorial at gabay para sa mga user  
- üìú [MCP Specification](https://spec.modelcontextprotocol.io/) ‚Äì Arkitektura ng protocol at teknikal na mga reperensya  
- üßë‚Äçüíª [MCP GitHub Repository](https://github.com/modelcontextprotocol) ‚Äì Open-source na SDKs, mga tools, at mga sample ng code
