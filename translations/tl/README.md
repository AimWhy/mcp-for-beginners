## üß† Pangkalahatang-ideya ng Kurikulum ng Model Context Protocol

Ang **Model Context Protocol (MCP)** ay isang makabagong balangkas na nilikha upang gawing pare-pareho ang pakikipag-ugnayan sa pagitan ng mga AI model at mga client application. Ang open-source na kurikulum na ito ay nag-aalok ng maayos na landas sa pag-aaral, kumpleto sa mga praktikal na halimbawa ng coding at mga totoong gamit, sa mga sikat na programming language tulad ng C#, Java, JavaScript, TypeScript, at Python.

Kung ikaw man ay isang AI developer, system architect, o software engineer, ang gabay na ito ang iyong komprehensibong sanggunian para maunawaan ang mga pundasyon at mga estratehiya sa pagpapatupad ng MCP.

## üîó Opisyal na Mga Mapagkukunan ng MCP

- üìò [MCP Documentation](https://modelcontextprotocol.io/) ‚Äì Detalyadong mga tutorial at gabay para sa mga gumagamit  
- üìú [MCP Specification](https://spec.modelcontextprotocol.io/) ‚Äì Arkitektura ng protocol at mga teknikal na sanggunian
