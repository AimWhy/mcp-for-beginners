# 🚀 מדריך מקיף לתוכנית הלימודים של פרוטוקול הקשר של המודל (MCP) למתחילים

## **למדו MCP עם דוגמאות קוד מעשיות ב-C#, Java, JavaScript, Python ו-TypeScript**

## 🧠 סקירה כללית של תוכנית הלימודים לפרוטוקול הקשר של המודל

**Model Context Protocol (MCP)** הוא מסגרת מתקדמת שנועדה לאחד את האינטראקציות בין מודלים של בינה מלאכותית ליישומי לקוח. תוכנית הלימודים הקוד הפתוח הזו מציעה מסלול לימודי מובנה, כולל דוגמאות קוד מעשיות ומקרי שימוש אמיתיים, בשפות תכנות פופולריות כמו C#, Java, JavaScript, TypeScript ו-Python.

בין אם אתם מפתחים בתחום ה-AI, אדריכלי מערכות או מהנדסי תוכנה, מדריך זה הוא המשאב המקיף שלכם לשליטה ביסודות MCP ואסטרטגיות יישום.

## 🔗 משאבים רשמיים של MCP

- 📘 [MCP Documentation](https://modelcontextprotocol.io/) – מדריכים מפורטים ומדריכי משתמש  
- 📜 [MCP Specification](https://spec.modelcontextprotocol.io/) – ארכיטקטורת הפרוטוקול והפניות טכניות  
- 🧑‍💻 [MCP GitHub Repository](https://github.com/modelcontextprotocol) – SDKs בקוד פתוח, כלים ודוגמאות קוד
