# ਇਸ ਨਮੂਨੇ ਨੂੰ ਚਲਾਉਣਾ

## -1- Dependencies ਇੰਸਟਾਲ ਕਰੋ

```bash
npm install
```

## -3- ਸਰਵਰ ਚਲਾਓ

```bash
npm run build
```

## -4- ਕਲਾਇੰਟ ਚਲਾਓ

```sh
npm run client
```
