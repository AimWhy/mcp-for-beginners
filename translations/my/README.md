# 🚀 Model Context Protocol (MCP) အခြေခံသင်ရိုးညွှန်းတမ်း

## **C#, Java, JavaScript, Python, နှင့် TypeScript တို့ဖြင့် MCP ကို လက်တွေ့ ကုဒ်နမူနာများဖြင့် သင်ယူပါ**

## 🧠 Model Context Protocol သင်ရိုးညွှန်းတမ်း အကြောင်းအနှစ်ချုပ်

**Model Context Protocol (MCP)** သည် AI မော်ဒယ်များနှင့် ဖောက်သည်အပလီကေးရှင်းများအကြား အပြန်အလှန် ဆက်သွယ်မှုကို စံသတ်မှတ်ပေးရန် ရည်ရွယ်ထားသည့် နောက်ဆုံးပေါ် ဖရိမ်ဝေါ့ခ်တစ်ခုဖြစ်သည်။ ဒီ open-source သင်ရိုးညွှန်းတမ်းတွင် C#, Java, JavaScript, TypeScript နှင့် Python ကဲ့သို့ လူကြိုက်များသော ပရိုဂရမ်မင်းဘာသာစကားများဖြင့် လက်တွေ့ ကုဒ်နမူနာများနှင့် အမှန်တကယ် အသုံးချနိုင်သော အခြေအနေများ ပါဝင်သော တစ်စဉ်တစ်ရာ သင်ယူမှုလမ်းကြောင်း တစ်ခုကို ပေးထားသည်။

သင်သည် AI ဖွံ့ဖြိုးသူ၊ စနစ်ဒီဇိုင်နာ သို့မဟုတ် ဆော့ဖ်ဝဲအင်ဂျင်နီယာ ဖြစ်စေ၊ ဒီလမ်းညွှန်သည် MCP အခြေခံများနှင့် အသုံးပြုမှုနည်းလမ်းများကို ကျွမ်းကျင်စွာ လေ့လာရန် လိုအပ်သည့် လမ်းညွှန်စာအုပ်ဖြစ်သည်။

## 🔗 MCP တရားဝင် အရင်းအမြစ်များ

- 📜 [MCP Specification](https://spec.modelcontextprotocol.io/) – ပရိုတိုကော ဖွဲ့စည်းမှုနှင့် နည်းပညာဆိုင်ရာ ကိုးကားချက်များ  
- 🧑‍💻 [MCP GitHub Repository](https://github.com/modelcontextprotocol) – Open-source SDK များ၊ ကိရိယာများနှင့် ကုဒ်နမူနာများ
