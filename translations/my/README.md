# 🚀 Model Context Protocol (MCP) အခြေခံသင်ရိုး

## **C#, Java, JavaScript, Python, နှင့် TypeScript ဖြင့် MCP ကို လက်တွေ့ကုဒ်နမူနာများဖြင့် သင်ယူပါ**

## 🧠 Model Context Protocol သင်ရိုး၏ အနှစ်ချုပ်

**Model Context Protocol (MCP)** သည် AI မော်ဒယ်များနှင့် client application များအကြား ဆက်သွယ်မှုများကို စံချိန်စနစ်တကျ စီမံရန် ရည်ရွယ်ထားသော နောက်ဆုံးပေါ် framework ဖြစ်သည်။ ဒီ open-source သင်ရိုးမှာ C#, Java, JavaScript, TypeScript နှင့် Python ကဲ့သို့ လူကြိုက်များသော programming ဘာသာစကားများတွင် လက်တွေ့ကုဒ်နမူနာများနှင့် အမှန်တကယ်အသုံးချမှုများပါဝင်ပြီး စနစ်တကျ သင်ကြားမှုလမ်းကြောင်းတစ်ခုကို ပေးစွမ်းသည်။

သင်သည် AI developer, system architect သို့မဟုတ် software engineer ဖြစ်ပါစေ၊ ဒီလမ်းညွှန်စာအုပ်သည် MCP အခြေခံများနှင့် အကောင်အထည်ဖော်နည်းများကို ကျွမ်းကျင်စွာ သင်ယူနိုင်ရန် အပြည့်အစုံသော အရင်းအမြစ်ဖြစ်သည်။

## 🔗 MCP အတည်ပြု အရင်းအမြစ်များ

- 📘 [MCP Documentation](https://modelcontextprotocol.io/) – အသေးစိတ် သင်ခန်းစာများနှင့် အသုံးပြုသူလမ်းညွှန်များ  
- 📜 [MCP Specification](https://spec.modelcontextprotocol.io/) – protocol architecture နှင့် နည်းပညာဆိုင်ရာ ရင်းမြစ်များ  
- 🧑‍💻 [MCP GitHub Repository](https://github.com/modelcontextprotocol) – open-source SDK များ၊ ကိရိယာများနှင့် ကုဒ်နမူနာများ
