# 🚀 Model Context Protocol (MCP) အခြေခံသင်တန်း

## **C#, Java, JavaScript, Python, နှင့် TypeScript မှာ လက်တွေ့ကုဒ်နမူနာတွေနဲ့ MCP ကို သင်ယူပါ**

## 🧠 Model Context Protocol သင်ရိုးညွှန်းတမ်း အနှစ်ချုပ်

**Model Context Protocol (MCP)** သည် AI မော်ဒယ်များနှင့် client application များအကြား အပြန်အလှန်ဆက်သွယ်မှုများကို စံသတ်မှတ်ရန် ရည်ရွယ်ထားသော နောက်ဆုံးပေါ် framework ဖြစ်သည်။ ဤ open-source သင်ရိုးညွှန်းတမ်းတွင် C#, Java, JavaScript, TypeScript နှင့် Python အပါအဝင် လူကြိုက်များသော programming ဘာသာစကားများဖြင့် လက်တွေ့ကုဒ်နမူနာများနှင့် အမှန်တကယ်အသုံးပြုနိုင်သော အကြောင်းအရာများ ပါဝင်သည်။

သင်သည် AI ဖန်တီးသူ၊ စနစ်ဒီဇိုင်နာ သို့မဟုတ် software engineer ဖြစ်ပါစေ၊ ဤလမ်းညွှန်သည် MCP အခြေခံအကြောင်းအရာများနှင့် အသုံးချနည်းများကို ကျွမ်းကျင်စေရန် အပြည့်အစုံသော အရင်းအမြစ်ဖြစ်ပါသည်။

## 🔗 MCP တရားဝင် အရင်းအမြစ်များ

- 📘 [MCP စာရွက်စာတမ်းများ](https://modelcontextprotocol.io/) – အသေးစိတ် သင်ခန်းစာများနှင့် အသုံးပြုသူလမ်းညွှန်များ  
- 📜 [MCP သတ်မှတ်ချက်](https://spec.modelcontextprotocol.io/) – Protocol ဖွဲ့စည်းပုံနှင့် နည်းပညာဆိုင်ရာ ရည်ညွှန်းချက်များ  
- 🧑‍💻 [MCP GitHub Repository](https://github.com/modelcontextprotocol) – Open-source SDK များ၊ ကိရိယာများနှင့် ကုဒ်နမူနာများ
