# ဒီနမူနာကို chạy ဖို့

## -1- လိုအပ်သော dependencies တွေကို 설치 လုပ်ပါ

```bash
dotnet restore
```

## -3- နမူနာကို chạy လုပ်ပါ

```bash
dotnet run
```
