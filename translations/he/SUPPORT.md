# :משימות בעל המאגר עדיין לא ערך את הקובץ הזה

**בעל המאגר**: האם אתה רוצה תמיכה של שירות לקוחות ותמיכה (CSS) עבור מוצר/פרויקט זה?

- **אין תמיכת CSS:** מלא את התבנית הזו עם מידע על איך להגיש בעיות ולקבל עזרה.
- **כן, תמיכת CSS:** מלא טופס קבלה בכתובת [aka.ms/onboardsupport](https://aka.ms/onboardsupport). CSS יעבוד איתך/יעזור לך לקבוע את הצעדים הבאים.
- **לא בטוח?** מלא טופס קבלה כאילו התשובה הייתה "כן". CSS יעזור לך להחליט.

*לאחר מכן הסר את הכותרת הראשונה מקובץ SUPPORT.MD לפני פרסום המאגר שלך.*

# תמיכה

## איך להגיש בעיות ולקבל עזרה

פרויקט זה משתמש ב-GitHub Issues כדי לעקוב אחר באגים ובקשות לפיצ'רים. אנא חפש את הבעיות הקיימות לפני הגשת בעיות חדשות כדי להימנע מכפילויות. עבור בעיות חדשות, הגש את הבאג שלך או בקשת הפיצ'ר כבעיה חדשה.

לעזרה ושאלות לגבי שימוש בפרויקט זה, אנא **מנהל המאגר: הכנס כאן הוראות כיצד ליצור קשר עם בעלי המאגר או הקהילה לקבלת עזרה. יכול להיות תג ב-Stack Overflow או ערוץ אחר. איפה תעזור לאנשים?**.
