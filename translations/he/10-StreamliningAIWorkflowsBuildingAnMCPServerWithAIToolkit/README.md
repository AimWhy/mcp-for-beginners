## 🎯 סקירה כללית

ברוכים הבאים ל**סדנת Model Context Protocol (MCP)**! סדנה מעשית ומקיפה זו משלבת שתי טכנולוגיות פורצות דרך לשינוי הדרך שבה מפתחים יישומי AI:

- **🔗 Model Context Protocol (MCP)**: תקן פתוח לשילוב חלק בין כלי AI
- **🛠️ AI Toolkit ל-Visual Studio Code (AITK)**: התוסף החזק של מיקרוסופט לפיתוח AI

### 🎓 מה תלמדו

בסיום הסדנה, תשלטו ביצירת יישומים חכמים שמקשרים בין מודלים של AI לכלים ושירותים מהעולם האמיתי. מבדיקות אוטומטיות ועד אינטגרציות API מותאמות אישית, תרכשו מיומנויות מעשיות לפתרון אתגרים עסקיים מורכבים.

## 🏗️ ערכת טכנולוגיות

### 🔌 Model Context Protocol (MCP)

MCP הוא ה**"USB-C של ה-AI"** – תקן אוניברסלי שמחבר מודלי AI לכלים חיצוניים ומקורות מידע.

**✨ תכונות מרכזיות:**
- 🔄 **אינטגרציה סטנדרטית**: ממשק אחיד לחיבור כלים ל-AI
- 🏛️ **ארכיטקטורה גמישה**: שרתים מקומיים ומרוחקים באמצעות stdio/SSE
- 🧰 **אקוסיסטם עשיר**: כלים, פרומפטים ומשאבים בפרוטוקול אחד
- 🔒 **מוכן לארגונים**: אבטחה ואמינות מובנות

**🎯 למה MCP חשוב:**
בדומה ל-USB-C שסיים את כאוס הכבלים, MCP מפשט את המורכבות של אינטגרציות AI. פרוטוקול אחד, אפשרויות בלתי מוגבלות.

### 🤖 AI Toolkit ל-Visual Studio Code (AITK)

התוסף המוביל של מיקרוסופט לפיתוח AI, שהופך את VS Code למרכז עוצמתי של AI.

**🚀 יכולות מרכזיות:**
- 📦 **קטלוג מודלים**: גישה למודלים מ-Azure AI, GitHub, Hugging Face, Ollama
- ⚡ **הסקה מקומית**: ביצוע אופטימלי CPU/GPU/NPU עם ONNX
- 🏗️ **בנאי סוכנים**: פיתוח סוכני AI ויזואלי עם אינטגרציה ל-MCP
- 🎭 **רב-מודאלי**: תמיכה בטקסט, חזון ופלט מובנה

**💡 יתרונות בפיתוח:**
- פריסת מודלים ללא קונפיגורציה
- הנדסת פרומפטים ויזואלית
- סביבת בדיקות בזמן אמת
- אינטגרציה חלקה עם שרת MCP

## 💡 יישומים והשפעות בעולם האמיתי

### 🏢 מקרים עסקיים בארגונים

#### 🔄 אוטומציה ב-DevOps
הפכו את זרימת העבודה שלכם לאוטומטית וחכמה:
- **ניהול מאגר חכם**: סקירת קוד והחלטות מיזוג מונעות AI
- **CI/CD אינטיליגנטי**: אופטימיזציה אוטומטית של פייפליין בהתבסס על שינויים בקוד
- **מיון תקלות**: סיווג והקצאת באגים אוטומטית

#### 🧪 מהפכת אבטחת איכות
שדרגו בדיקות עם אוטומציה מבוססת AI:
- **יצירת מבחנים חכמה**: הפקה אוטומטית של מערכי מבחנים מקיפים
- **בדיקות רגרסיה ויזואליות**: זיהוי שינויים בממשק משתמש מונחה AI
- **מעקב ביצועים**: זיהוי ופתרון בעיות בזמן אמת

#### 📊 אינטיליגנציה בצנרת נתונים
בנו זרימות עיבוד נתונים חכמות:
- **תהליכי ETL אדפטיביים**: טרנספורמציות נתונים שמשתפרות עצמאית
- **זיהוי חריגות**: ניטור איכות נתונים בזמן אמת
- **ניתוב חכם**: ניהול זרימת נתונים חכמה

#### 🎧 שיפור חווית הלקוח
צרו אינטראקציות יוצאות דופן עם הלקוחות:
- **תמיכה בהקשר**: סוכני AI עם גישה להיסטוריית הלקוח
- **פתרון בעיות יזום**: שירות לקוחות חיזוי
- **אינטגרציה רב-ערוצית**: חוויית AI מאוחדת בפלטפורמות שונות

## 🛠️ דרישות מוקדמות והתקנה

### 💻 דרישות מערכת

| רכיב                | דרישה               | הערות                  |
|---------------------|---------------------|------------------------|
| **מערכת הפעלה**     | Windows 10+, macOS 10.15+, Linux | כל מערכת מודרנית      |
| **Visual Studio Code** | גרסה יציבה אחרונה  | דרוש ל-AITK            |
| **Node.js**          | v18.0+ ו-npm        | לפיתוח שרת MCP         |
| **Python**           | 3.10+               | אופציונלי לשרתי MCP בפייתון |
| **זיכרון**           | מינימום 8GB RAM     | מומלץ 16GB למודלים מקומיים |

### 🔧 סביבת פיתוח

#### תוספי VS Code מומלצים
- **AI Toolkit** (ms-windows-ai-studio.windows-ai-studio)
- **Python** (ms-python.python)
- **Python Debugger** (ms-python.debugpy)
- **GitHub Copilot** (GitHub.copilot) - אופציונלי אך מועיל

#### כלים אופציונליים
- **uv**: מנהל חבילות פייתון מודרני
- **MCP Inspector**: כלי ניפוי שגיאות ויזואלי לשרתי MCP
- **Playwright**: דוגמאות לאוטומציה של דפדפן
