# 🚀 תכנית הלימודים של פרוטוקול הקשר למודל (MCP) למתחילים

## **למדו MCP עם דוגמאות קוד מעשיות ב-C#, Java, JavaScript, Python ו-TypeScript**

## 🧠 סקירה כללית של תכנית הלימודים של פרוטוקול הקשר למודל

**פרוטוקול הקשר למודל (MCP)** הוא מסגרת מתקדמת שנועדה לסטנדרטיזציה של האינטראקציות בין מודלי AI ליישומי לקוח. תכנית הלימודים בקוד פתוח זו מציעה מסלול למידה מובנה, הכולל דוגמאות קוד מעשיות ומקרי שימוש מהעולם האמיתי, בשפות תכנות פופולריות כגון C#, Java, JavaScript, TypeScript ו-Python.

בין אם אתה מפתח AI, אדריכל מערכות או מהנדס תוכנה, מדריך זה הוא המשאב המקיף שלך לשליטה ביסודות MCP ובאסטרטגיות יישום.

## 🔗 משאבים רשמיים של MCP

- 📘 [תיעוד MCP](https://modelcontextprotocol.io/) – מדריכים מפורטים ומדריכי משתמש  
- 📜 [מפרט MCP](https://spec.modelcontextprotocol.io/) – ארכיטקטורת הפרוטוקול והפניות טכניות  
- 🧑‍💻 [מאגר GitHub של MCP](https://github.com/modelcontextprotocol) – SDKs בקוד פתוח, כלים ודוגמאות קוד  

## הצטרפו אלינו ל-MCP Dev Days ב-29-30 ביולי 2025

התכוננו לשני ימים של תובנות טכניות מעמיקות, חיבור לקהילה ולמידה מעשית באירוע וירטואלי המוקדש לפרוטוקול הקשר למודל (MCP) — התקן המתפתח שמגשר בין מודלי AI לכלים שהם תלויים בהם.

➡️ [הרשמה ל-MCP Dev Days](https://developer.microsoft.com/en-us/reactor/series/S-1563/)

ניתן לצפות ב-MCP Dev Days על ידי הרשמה בדף האירוע שלנו: https://aka.ms/mcpdevdays. משם תוכלו להצטרף לשידור חי ביוטיוב או בטוויץ'. כל התכנים מוקלטים ויהיו זמינים לאחר מכן בערוץ היוטיוב של Microsoft Developer. קוד המקור של הדגמות יהיה זמין גם ב-GitHub.

#### יום 1: פרודוקטיביות MCP, כלי פיתוח וקהילה

היום מוקדש להעצמת מפתחים להשתמש ב-MCP במסגרת העבודה שלהם ולחגוג את קהילת MCP המדהימה. יצטרפו אלינו חברי קהילה ושותפים כמו Arcade, Block, Okta ו-Neon כדי לראות איך הם משתפים פעולה עם מיקרוסופט לעיצוב אקוסיסטם פתוח ומורחב של MCP. דגימות מהעולם האמיתי ב-VS Code, Visual Studio, GitHub Copilot וכלי קהילה פופולריים  
זרימות עבודה מעשיות מונחות הקשר  
מפגשים ותובנות בהובלת הקהילה  
בין אם אתם רק מתחילים עם MCP או כבר בונים איתו, יום 1 ייתן השראה ותובנות מעשיות.

#### יום 2: בניית שרתי MCP בביטחון

היום מיועד לבוני MCP. נצלול לעומק לאסטרטגיות יישום ושיטות עבודה מומלצות ליצירת שרתי MCP ואינטגרציה של MCP בזרימות העבודה של AI שלכם.

### נושאים כוללים:

- בניית שרתי MCP ואינטגרציה שלהם עם חוויות סוכנים  
- פיתוח מונחה פרומפט  
- שיטות עבודה מומלצות לאבטחה  
- שימוש בבלוקים לבניית פונקציות, ACA, וניהול API  
- התאמה לרישום וכלים (1P + 3P)

אם אתם מפתחים, בוני כלים או אסטרטגים למוצרי AI, היום הזה מלא בתובנות שתצטרכו לבניית פתרונות MCP סקלאביליים, מאובטחים ומוכנים לעתיד.
