# 🚀 תכנית לימודים לפרוטוקול Model Context (MCP) למתחילים

## **למדו MCP עם דוגמאות קוד מעשיות ב-C#, Java, JavaScript, Python ו-TypeScript**

## 🧠 סקירה כללית של תכנית הלימודים לפרוטוקול Model Context

**Model Context Protocol (MCP)** הוא מסגרת מתקדמת שנועדה לאחד את האינטראקציות בין מודלים של בינה מלאכותית לאפליקציות לקוח. תכנית הלימודים בקוד פתוח הזו מציעה מסלול לימוד מובנה, כולל דוגמאות קוד מעשיות ומקרי שימוש מהעולם האמיתי, בשפות תכנות פופולריות כמו C#, Java, JavaScript, TypeScript ו-Python.

בין אם אתה מפתח AI, ארכיטקט מערכות או מהנדס תוכנה, מדריך זה הוא המשאב המקיף שלך לשליטה ביסודות MCP ואסטרטגיות יישום.

## 🔗 משאבים רשמיים של MCP

- 📘 [MCP Documentation](https://modelcontextprotocol.io/) – מדריכים מפורטים ומדריכי משתמש  
- 📜 [MCP Specification](https://spec.modelcontextprotocol.io/) – ארכיטקטורת הפרוטוקול והפניות טכניות  
- 🧑‍💻 [MCP GitHub Repository](https://github.com/modelcontextprotocol) – SDKs, כלים ודוגמאות קוד בקוד פתוח
