# 🚀 תכנית לימודים לפרוטוקול הקשר של מודלים (MCP) למתחילים

## **למדו MCP עם דוגמאות קוד מעשיות ב-C#, Java, JavaScript, Python ו-TypeScript**

## 🧠 סקירה כללית של תכנית הלימודים לפרוטוקול הקשר של מודלים

**Model Context Protocol (MCP)** הוא מסגרת מתקדמת שנועדה לאחד את האינטראקציות בין מודלים של בינה מלאכותית ליישומים של לקוחות. תכנית הלימודים בקוד פתוח הזו מציעה מסלול למידה מובנה, עם דוגמאות קוד מעשיות ומקרי שימוש מהעולם האמיתי, בשפות תכנות פופולריות כמו C#, Java, JavaScript, TypeScript ו-Python.

בין אם אתם מפתחי בינה מלאכותית, אדריכלי מערכות או מהנדסי תוכנה, מדריך זה הוא המשאב המלא שלכם לשליטה ביסודות MCP ואסטרטגיות יישום.

## 🔗 משאבים רשמיים של MCP

- 📘 [MCP Documentation](https://modelcontextprotocol.io/) – מדריכים מפורטים ומדריכי משתמש  
- 📜 [MCP Specification](https://spec.modelcontextprotocol.io/) – ארכיטקטורת הפרוטוקול והפניות טכניות  
- 🧑‍💻 [MCP GitHub Repository](https://github.com/modelcontextprotocol) – SDKs בקוד פתוח, כלים ודוגמאות קוד
