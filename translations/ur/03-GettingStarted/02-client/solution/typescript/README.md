# اس نمونے کو چلانا

آپ کو `uv` انسٹال کرنے کی سفارش کی جاتی ہے، لیکن یہ ضروری نہیں ہے، دیکھیں [ہدایات](https://docs.astral.sh/uv/#highlights)

## -1- ضروریات انسٹال کریں

```bash
npm install
```

## -3- سرور چلائیں

```bash
npm run build
```

## -4- کلائنٹ چلائیں

```sh
npm run client
```
