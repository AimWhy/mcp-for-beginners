# 🚀 ماڈل کانٹیکسٹ پروٹوکول (MCP) نصاب برائے مبتدیان

## **C#، Java، JavaScript، Python، اور TypeScript میں عملی کوڈ مثالوں کے ساتھ MCP سیکھیں**

## 🧠 ماڈل کانٹیکسٹ پروٹوکول نصاب کا جائزہ

**Model Context Protocol (MCP)** ایک جدید فریم ورک ہے جو AI ماڈلز اور کلائنٹ ایپلیکیشنز کے درمیان تعاملات کو معیاری بنانے کے لیے ڈیزائن کیا گیا ہے۔ یہ اوپن سورس نصاب ایک منظم تعلیمی راستہ فراہم کرتا ہے، جس میں عملی کوڈنگ مثالیں اور حقیقی دنیا کے استعمال کے کیسز شامل ہیں، جو مقبول پروگرامنگ زبانوں جیسے C#، Java، JavaScript، TypeScript، اور Python پر محیط ہیں۔

چاہے آپ AI ڈیولپر ہوں، سسٹم آرکیٹیکٹ، یا سافٹ ویئر انجینئر، یہ رہنما MCP کے بنیادی اصول اور نفاذ کی حکمت عملیوں میں مہارت حاصل کرنے کے لیے آپ کا جامع ذریعہ ہے۔

## 🔗 MCP کے سرکاری وسائل

- 📘 [MCP Documentation](https://modelcontextprotocol.io/) – تفصیلی ٹیوٹوریلز اور صارف رہنما  
- 📜 [MCP Specification](https://spec.modelcontextprotocol.io/) – پروٹوکول کی ساخت اور تکنیکی حوالہ جات  
- 🧑‍💻 [MCP GitHub Repository](https://github.com/modelcontextprotocol) – اوپن سورس SDKs، ٹولز، اور کوڈ مثالیں
