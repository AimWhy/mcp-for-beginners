# 🚀 ماڈل کانٹیکسٹ پروٹوکول (MCP) نصاب برائے مبتدیان

## **C#، Java، JavaScript، Python، اور TypeScript میں عملی کوڈ مثالوں کے ساتھ MCP سیکھیں**

## 🧠 ماڈل کانٹیکسٹ پروٹوکول نصاب کا جائزہ

**Model Context Protocol (MCP)** ایک جدید فریم ورک ہے جو AI ماڈلز اور کلائنٹ ایپلیکیشنز کے درمیان تعاملات کو معیاری بنانے کے لیے ڈیزائن کیا گیا ہے۔ یہ اوپن سورس نصاب ایک منظم تعلیمی راستہ فراہم کرتا ہے، جس میں عملی کوڈنگ کی مثالیں اور حقیقی دنیا کے استعمال کے کیسز شامل ہیں، جو C#، Java، JavaScript، TypeScript، اور Python سمیت مقبول پروگرامنگ زبانوں میں ہیں۔

چاہے آپ AI ڈیولپر ہوں، سسٹم آرکیٹیکٹ، یا سافٹ ویئر انجینئر، یہ رہنما MCP کے بنیادی اصولوں اور نفاذ کی حکمت عملیوں میں مہارت حاصل کرنے کے لیے آپ کا جامع ذریعہ ہے۔

## 🔗 سرکاری MCP وسائل

- 📘 [MCP Documentation](https://modelcontextprotocol.io/) – تفصیلی ٹیوٹوریلز اور صارف گائیڈز  
- 📜 [MCP Specification](https://spec.modelcontextprotocol.io/) – پروٹوکول کی ساخت اور تکنیکی حوالہ جات  
- 🧑‍💻 [MCP GitHub Repository](https://github.com/modelcontextprotocol) – اوپن سورس SDKs، ٹولز، اور کوڈ کے نمونے
