# 🚀 ماڈل کانٹیکسٹ پروٹوکول (MCP) نصاب برائے ابتدائی افراد

## **C#، Java، JavaScript، Python، اور TypeScript میں عملی کوڈ کی مثالوں کے ساتھ MCP سیکھیں**

## 🧠 ماڈل کانٹیکسٹ پروٹوکول نصاب کا جائزہ

**ماڈل کانٹیکسٹ پروٹوکول (MCP)** ایک جدید فریم ورک ہے جو AI ماڈلز اور کلائنٹ ایپلیکیشنز کے درمیان تعاملات کو معیاری بنانے کے لیے تیار کیا گیا ہے۔ یہ اوپن سورس نصاب ایک منظم تعلیمی راستہ فراہم کرتا ہے، جس میں عملی کوڈنگ مثالیں اور حقیقی دنیا کے استعمال کے کیسز شامل ہیں، جو مقبول پروگرامنگ زبانوں جیسے C#، Java، JavaScript، TypeScript، اور Python میں دستیاب ہیں۔

چاہے آپ AI ڈیویلپر ہوں، سسٹم آرکیٹیکٹ ہوں یا سافٹ ویئر انجینئر، یہ رہنما MCP کی بنیادی باتوں اور نفاذ کی حکمت عملیوں میں مہارت حاصل کرنے کے لیے آپ کا جامع ذریعہ ہے۔

## 🔗 سرکاری MCP وسائل

- 📘 [MCP دستاویزات](https://modelcontextprotocol.io/) – تفصیلی ٹیوٹوریلز اور صارف گائیڈز  
- 📜 [MCP وضاحت](https://spec.modelcontextprotocol.io/) – پروٹوکول کی ساخت اور تکنیکی حوالہ جات  
- 🧑‍💻 [MCP GitHub ریپوزیٹری](https://github.com/modelcontextprotocol) – اوپن سورس SDKs، ٹولز، اور کوڈ نمونے
