## Intégration de MCP avec Azure ML

L'intégration de MCP avec Azure Machine Learning (ML) vous permet de tirer parti des puissantes capacités de ML d'Azure tout en conservant la flexibilité de MCP. Cette intégration peut être utilisée pour exécuter des pipelines ML, enregistrer des modèles en tant qu'outils et gérer les ressources de calcul.
