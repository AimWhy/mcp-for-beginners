## Интеграция на MCP с Azure ML

Интегрирането на MCP с Azure Machine Learning (ML) ви позволява да използвате мощните ML възможности на Azure, като същевременно запазвате гъвкавостта на MCP. Тази интеграция може да се използва за изпълнение на ML пайплайни, регистриране на модели като инструменти и управление на изчислителни ресурси.
