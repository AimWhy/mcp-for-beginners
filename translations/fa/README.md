# 🚀 راهنمای جامع پروتکل مدل کانتکست (MCP) برای مبتدیان

## **یادگیری MCP با مثال‌های عملی کدنویسی در C#، Java، JavaScript، Python و TypeScript**

## 🧠 مرور کلی برنامه درسی پروتکل مدل کانتکست

**پروتکل مدل کانتکست (MCP)** یک چارچوب پیشرفته است که برای استانداردسازی تعاملات بین مدل‌های هوش مصنوعی و برنامه‌های مشتری طراحی شده است. این برنامه درسی متن‌باز، مسیر یادگیری ساختاریافته‌ای را همراه با مثال‌های عملی کدنویسی و کاربردهای واقعی در زبان‌های برنامه‌نویسی محبوب مانند C#، Java، JavaScript، TypeScript و Python ارائه می‌دهد.

چه توسعه‌دهنده هوش مصنوعی، معمار سیستم یا مهندس نرم‌افزار باشید، این راهنما منبع جامعی برای تسلط بر اصول پایه MCP و استراتژی‌های پیاده‌سازی آن است.

## 🔗 منابع رسمی MCP

- 📘 [مستندات MCP](https://modelcontextprotocol.io/) – آموزش‌ها و راهنمای کاربران به صورت مفصل  
- 📜 [مشخصات MCP](https://spec.modelcontextprotocol.io/) – معماری پروتکل و مراجع فنی  
- 🧑‍💻 [مخزن GitHub MCP](https://github.com/modelcontextprotocol) – SDKها، ابزارها و نمونه کدهای متن‌باز
