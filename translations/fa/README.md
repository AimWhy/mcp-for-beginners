# 🚀 برنامه درسی پروتکل مدل کانتکست (MCP) برای مبتدی‌ها

## **یادگیری MCP با مثال‌های عملی کدنویسی در C#، Java، JavaScript، Python و TypeScript**

## 🧠 مروری بر برنامه درسی پروتکل مدل کانتکست

**پروتکل مدل کانتکست (MCP)** چارچوبی پیشرفته است که برای استانداردسازی تعاملات بین مدل‌های هوش مصنوعی و برنامه‌های کلاینت طراحی شده است. این برنامه درسی متن‌باز، مسیر یادگیری ساختاریافته‌ای را با مثال‌های عملی کدنویسی و موارد استفاده واقعی در زبان‌های برنامه‌نویسی محبوب مانند C#، Java، JavaScript، TypeScript و Python ارائه می‌دهد.

چه توسعه‌دهنده هوش مصنوعی باشید، چه معمار سیستم یا مهندس نرم‌افزار، این راهنما منبع جامع شما برای تسلط بر اصول و استراتژی‌های پیاده‌سازی MCP است.

## 🔗 منابع رسمی MCP

- 📘 [مستندات MCP](https://modelcontextprotocol.io/) – آموزش‌ها و راهنماهای جامع  
- 📜 [مشخصات MCP](https://spec.modelcontextprotocol.io/) – معماری پروتکل و مراجع فنی  
- 🧑‍💻 [مخزن گیت‌هاب MCP](https://github.com/modelcontextprotocol) – SDKها، ابزارها و نمونه کدهای متن‌باز
