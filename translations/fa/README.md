# 🚀 برنامه درسی پروتکل مدل کانتکست (MCP) برای مبتدیان

## **یادگیری MCP با مثال‌های عملی کدنویسی در C#، Java، JavaScript، Python و TypeScript**

## 🧠 مروری بر برنامه درسی پروتکل مدل کانتکست

**پروتکل مدل کانتکست (MCP)** یک چارچوب پیشرفته است که برای استانداردسازی تعاملات بین مدل‌های هوش مصنوعی و برنامه‌های کلاینت طراحی شده است. این برنامه درسی متن‌باز، مسیر یادگیری ساختارمندی را ارائه می‌دهد که همراه با مثال‌های عملی کدنویسی و موارد کاربرد واقعی در زبان‌های برنامه‌نویسی محبوب مانند C#، Java، JavaScript، TypeScript و Python است.

چه توسعه‌دهنده هوش مصنوعی باشید، چه معمار سیستم یا مهندس نرم‌افزار، این راهنما منبع جامع شما برای تسلط بر اصول و استراتژی‌های پیاده‌سازی MCP است.

## 🔗 منابع رسمی MCP

- 📘 [مستندات MCP](https://modelcontextprotocol.io/) – آموزش‌ها و راهنمای کاربران به‌صورت جامع
