## مثال چندرسانه‌ای: پردازش صدا

پردازش صدا یکی دیگر از حالت‌های رایج در برنامه‌های چندرسانه‌ای است. در ادامه نمونه‌ای از پیاده‌سازی ابزاری برای رونویسی صوتی آورده شده است که می‌تواند فایل‌های صوتی را پردازش کرده و متن رونویسی شده را بازگرداند.
