# اجرای این نمونه

توصیه می‌شود که `uv` را نصب کنید، اما این کار ضروری نیست. به [دستورالعمل‌ها](https://docs.astral.sh/uv/#highlights) مراجعه کنید.

## -1- وابستگی‌ها را نصب کنید

```bash
npm install
```

## -3- سرور را اجرا کنید

```bash
npm run build
```

## -4- مشتری را اجرا کنید

```sh
npm run client
```

شما باید نتیجه‌ای مشابه با این ببینید:

```text
Prompt:  {
  type: 'text',
  text: 'Please review this code:\n\nconsole.log("hello");'
}
Resource template:  file
Tool result:  { content: [ { type: 'text', text: '9' } ] }
```
