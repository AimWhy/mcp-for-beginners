## Consume the features in the server

- Click the `play` icon, once you've added *mcp.json* to *./vscode* folder,

    Observe the tooling icon change to increase the number of available tools. Tooling icon is located right above the chat field in GitHub Copilot.

## Run a tool

- Type a prompt in your chat window that matches the description of your tool. For example to trigger the tool `add`, majd beírnál valamit, például "add 3 to 20".

    Látnod kell egy eszközt megjelenni a csevegő szövegdoboz fölött, amely arra kér, hogy válaszd ki az eszközt a futtatáshoz, ahogy az alábbi képen látható:

    ![VS Code jelzi, hogy eszközt szeretne futtatni](../../../../../translated_images/vscode-agent.d5a0e0b897331060518fe3f13907677ef52b879db98c64d68a38338608f3751e.hu.png)

    Az eszköz kiválasztása egy numerikus eredményt kell, hogy adjon, például "23"-at, ha az előzőleg említett promptot használtad.
