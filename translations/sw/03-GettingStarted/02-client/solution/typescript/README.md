# Kuendesha sampuli hii

Inapendekezwa usakinishe `uv` lakini si lazima, angalia [maelekezo](https://docs.astral.sh/uv/#highlights)

## -1- Sakinisha utegemezi

```bash
npm install
```

## -3- Endesha seva

```bash
npm run build
```

## -4- Endesha mteja

```sh
npm run client
```
