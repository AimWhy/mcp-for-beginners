## Інтеграція MCP з Azure ML

Інтеграція MCP з Azure Machine Learning (ML) дозволяє використовувати потужні можливості ML Azure, зберігаючи при цьому гнучкість MCP. Цю інтеграцію можна застосовувати для виконання ML-пайплайнів, реєстрації моделей як інструментів та управління обчислювальними ресурсами.
