## Integrace MCP s Azure ML

Integrace MCP s Azure Machine Learning (ML) vám umožňuje využít výkonné ML schopnosti Azure při zachování flexibility MCP. Tato integrace může být použita k spouštění ML pipeline, registraci modelů jako nástrojů a správě výpočetních zdrojů.
