## Integrace MCP s Azure ML

Integrace MCP s Azure Machine Learning (ML) vám umožňuje využít výkonné ML schopnosti Azure a zároveň zachovat flexibilitu MCP. Tato integrace může být použita k provádění ML pipeline, registraci modelů jako nástrojů a správě výpočetních zdrojů.
